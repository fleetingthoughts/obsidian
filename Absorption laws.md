The absorption laws allow us to find an identity for a given set of interest, $A$ in relation to another arbitrary set $B$: $$A = A\cup (B\cap A)=A\cap (B \cup A)$$
The identity is useful as a sort of "strategic addition of zero" to substitute for any arbitrary set.
