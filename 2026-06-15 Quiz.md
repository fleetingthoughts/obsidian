## 🔑 Comprehensive Answer Key

## Monday

- M1: Let $V$ be a vector space over a field $F$. If $0$ and $0'$ are both additive identities, then $0 = 0 + 0' = 0'$. If $y$ and $y'$ are both additive inverses of $x \in V$, then $y = y + 0 = y + (x + y') = (y + x) + y' = 0 + y' = y'$.
    
    - _Source_: Linear Algebra (Friedberg, Insel, Spence) / Vector Space Axioms.
    
- M2: If a sequence $(a_n)$ of real numbers converges to $L_1$ and $L_2$, then for every $\epsilon > 0$, there exist $N_1, N_2$ such that $\vert{}a_n - L_1\vert{} < \frac{\epsilon}{2}$ for $n \ge N_1$ and $\vert{}a_n - L_2\vert{} < \frac{\epsilon}{2}$ for $n \ge N_2$. For $n \ge \max(N_1, N_2)$, $\vert{}L_1 - L_2\vert{} \le \vert{}L_1 - a_n\vert{} + \vert{}a_n - L_2\vert{} < \epsilon$. Since $\epsilon$ is arbitrary, $L_1 = L_2$.
    
    - _Source_: Understanding Analysis (Abbott) / Uniqueness of Limits.
    
- M3: A square matrix $A$ is skew-symmetric if and only if $A^t = -A$, meaning its transpose equals its negative. Consequently, its entry-wise elements satisfy $a_{ij} = -a_{ji}$, forcing all diagonal elements $a_{ii}$ to be 0.
    
    - _Source_: Linear Algebra and Its Applications (Lay) / Matrix Properties.
    
- M4: 1. Sample Space: Choose any 5 cards from 52: $\binom{52}{5}$. 2. Choose triple rank: $\binom{13}{1}$. 3. Choose triple suits: $\binom{4}{3}$. 4. Choose pair rank: $\binom{12}{1}$. 5. Choose pair suits: $\binom{4}{2}$. 6. Calculate ratio: $\left[\binom{13}{1}\binom{4}{3} \times \binom{12}{1}\binom{4}{2}\right] / \binom{52}{5}$.
    
    - _Source_: A First Course in Probability (Ross) / Combinatorial Analysis.
    
- M5: 1. Verify Injectivity: Assume $T(u) = T(v)$ and show $u = v$ algebraically (or prove $\text{N}(T) = \{0\}$). 2. Verify Surjectivity: Take any $w \in W$ and explicitly construct or prove the existence of a $v \in V$ such that $T(v) = w$ (or show $\dim(V) = \dim(W) = \text{rank}(T)$).
    
    - _Source_: Linear Algebra (Friedberg, Insel, Spence) / Isomorphism Characterization Theorem.
    
- M6: 1. Set up the characteristic equation: $\det(A - \lambda I) = 0$. 2. Compute the determinant to get the characteristic polynomial $p(\lambda)$. 3. Solve the polynomial roots for $\lambda$, yielding the algebraic spectrum of eigenvalues.
    
    - _Source_: Introduction to Linear Algebra (Strang) / Eigenvalue Problem.
    
- M7: Let $T: V \to W$ be an injective linear transformation. If $\sum_{i=1}^k c_i T(s_i) = 0$ for $s_i \in S$, by linearity $T(\sum_{i=1}^k c_i s_i) = 0$. Since $T$ is injective, $\sum_{i=1}^k c_i s_i = 0$. Because $S$ is linearly independent, $c_i = 0$ for all $i$. Hence, $T(S)$ is linearly independent.
    
    - _Source_: Linear Algebra (Friedberg, Insel, Spence) / Injection Properties of Transformations.
    
- M8: A set $F \subseteq \mathbb{R}$ is closed if and only if it contains all of its limit points. Equivalently, if every convergent sequence $(x_n)$ contained in $F$ has its limit $x = \lim x_n$ inside $F$.
    
    - _Source_: Understanding Analysis (Abbott) / Topology of $\mathbb{R}$.
    

## Tuesday

- T1: A set $P \subseteq \mathbb{R}$ is perfect if it is closed and contains no isolated points. That is, every point in $P$ is a limit point of $P$.
    
    - _Source_: Principles of Mathematical Analysis (Rudin) / Point-Set Topology.
    
- T2: If $\lim a_n = A$ and $\lim b_n = B$, then: $\lim(a_n+b_n) = A+B$; $\lim(c \cdot a_n) = cA$; $\lim(a_n \cdot b_n) = AB$; $\lim(a_n/b_n) = A/B$ provided $b_n \neq 0$ and $B \neq 0$.
    
    - _Source_: Understanding Analysis (Abbott) / Algebraic Limit Theorem.
    
- T3: A square matrix $M$ is a diagonal matrix if all its off-diagonal entries are zero. That is, $m_{ij} = 0$ whenever $i \neq j$.
    
    - _Source_: Matrix Analysis (Horn & Johnson) / Matrix Paradigms.
    
- T4: For any conformable matrices $A$ and $B$, $(AB)^t = B^t A^t$. The transpose of a product is the product of the transposes in reverse order.
    
    - _Source_: Introduction to Linear Algebra (Strang) / Transpose Properties.
    
- T5: Let $I_n = [a_n, b_n]$ be nested. The set of left endpoints $A = \{a_n : n \in \mathbb{N}\}$ is bounded above by any $b_m$. By completeness, $x = \sup A$ exists. Since $x$ is the least upper bound, $a_n \le x$ for all $n$. Because each $b_n$ is an upper bound for $A$, $x \le b_n$. Thus $x \in I_n$ for all $n$, meaning $\bigcap_{n=1}^\infty I_n \neq \emptyset$.
    
    - _Source_: Understanding Analysis (Abbott) / Nested Interval Postulate Proof.
    
- T6: $\text{Cov}(X,Y) = E[(X - E[X])(Y - E[Y])] = E[XY] - E[X]E[Y]$. If $X$ and $Y$ are independent, $E[XY] = E[X]E[Y]$, which simplifies the covariance to $0$.
    
    - _Source_: OpenIntro Statistics / Covariance Identities.
    
- T7: 1. Find the CDF of $Y$: $F_Y(y) = P(g(X) \le y)$. 2. If $g$ is increasing, $P(X \le g^{-1}(y)) = F_X(g^{-1}(y))$. 3. Differentiate with respect to $y$ via the chain rule to obtain the PDF: $f_Y(y) = f_X(g^{-1}(y)) \cdot \vert{}\frac{d}{dy} g^{-1}(y)\vert{}$.
    
    - _Source_: Mathematical Statistics with Applications (Wackerly) / Method of Transformations.
    
- T8: A collection of events $\{E_1, E_2, \dots, E_n\}$ is mutually exclusive if $E_i \cap E_j = \emptyset$ for all $i \neq j$, and exhaustive if $\bigcup_{i=1}^n E_i = S$. The unique statistical term for this is a Partition of the Sample Space.
    
    - _Source_: A First Course in Probability (Ross) / Foundations of Probability.
    

## Wednesday

- W1: 1. Linear Independence Check: Set $\sum c_i u_i = 0$ and verify the only solution is the trivial solution $c_i = 0$ (via row reduction to the identity). 2. Spanning Check: Verify that $\text{span}(\{u_1, \dots, u_k\}) = V$ by ensuring the number of independent vectors $k$ equals the known dimension $\dim(V)$.
    
    - _Source_: Linear Algebra and Its Applications (Lay) / Basis Tests.
    
- W2: The moment-generating function of a discrete random variable $X$ is defined as $M_X(t) = E[e^{tX}] = \sum_{x} e^{tx} \cdot p(x)$, for all values of $t$ where the series converges.
    
    - _Source_: Statistical Inference (Casella & Berger) / Moment Generating Functions.
    
- W3: If a matrix $B$ is obtained by multiplying exactly one row of an $n \times n$ matrix $A$ by a scalar $c$, then $\det(B) = c \cdot \det(A)$.
    
    - _Source_: Advanced Linear Algebra (Roman) / Determinant Functionals.
    
- W4: An $n \times n$ matrix $A$ is invertible if and only if $\text{rank}(A) = n$. This matches the condition that the matrix is of full rank, meaning its rows and columns are linearly independent.
    
    - _Source_: Introduction to Linear Algebra (Strang) / Invertibility Criteria.
    
- W5: For any matrix $A \in M_{m \times n}(F)$, $\text{rank}(A) = \text{rank}(A^t)$. The dimension of the row space equals the dimension of the column space.
    
    - _Source_: Linear Algebra (Friedberg, Insel, Spence) / Rank Theorems.
    
- W6: A non-empty subset $A \subseteq \mathbb{R}$ contains its supremum and infimum as maximum and minimum elements if and only if $A$ is a compact set (closed and bounded in $\mathbb{R}$).
    
    - _Source_: Principles of Mathematical Analysis (Rudin) / Heine-Borel Consequences.
    
- W7: An open cover of $A$ is a collection of open sets $\{G_\alpha\}$ such that $A \subseteq \bigcup G_\alpha$. A finite subcover is a finite subcollection $\{G_{\alpha_1}, G_{\alpha_2}, \dots, G_{\alpha_k}\}$ that still satisfies $A \subseteq \bigcup_{i=1}^k G_{\alpha_i}$.
    
    - _Source_: Understanding Analysis (Abbott) / Compactness.
    
- W8: For discrete random variables $X$ and $Y$, the conditional expectation is $E[X \mid Y=y] = \sum_{x} x \cdot P(X=x \mid Y=y) = \sum_{x} x \cdot \frac{P(X=x, Y=y)}{P(Y=y)}$, provided $P(Y=y) > 0$.
    
    - _Source_: Probability and Random Processes (Grimmett & Stirzaker) / Conditional Expectation.
    

## Thursday

- H1: $T^{-1}$ exists if and only if $T$ is a bijection. To prove linearity: Let $w_1, w_2 \in W$. Since $T$ is bijective, there exist unique $v_1, v_2 \in V$ such that $T(v_1)=w_1, T(v_2)=w_2$. Then $T^{-1}(w_1+w_2) = T^{-1}(T(v_1)+T(v_2)) = T^{-1}(T(v_1+v_2)) = v_1+v_2 = T^{-1}(w_1)+T^{-1}(w_2)$. Scalar multiplication follows symmetrically.
    
    - _Source_: Linear Algebra (Friedberg, Insel, Spence) / Inverse Transformations.
    
- H2: Let $\beta$ and $\beta'$ be ordered bases for $V$, and $\gamma$ and $\gamma'$ be ordered bases for $W$. Then $[T]_{\beta'}^{\gamma'} = P^{-1} [T]_\beta^\gamma Q$, where $P$ is the change-of-basis matrix from $\gamma'$ to $\gamma$, and $Q$ is the change-of-basis matrix from $\beta'$ to $\beta$.
    
    - _Source_: Linear Algebra (Friedberg, Insel, Spence) / Change of Basis Theorem.
    
- H3: $(\implies)$ If $A$ is invertible, $A A^{-1} = I \implies \det(A)\det(A^{-1}) = \det(I) = 1$, so $\det(A) \neq 0$. $(\impliedby)$ If $\det(A) \neq 0$, we can construct the inverse explicitly via the adjugate matrix: $A^{-1} = \frac{1}{\det(A)} \text{adj}(A)$.
    
    - _Source_: Advanced Linear Algebra (Roman) / Determinant Characterization.
    
- H4: A set is countable if there exists an injection into $\mathbb{N}$. For $\mathbb{Q}$, map fractions $\frac{p}{q}$ to an infinite 2D grid. Count along alternating anti-diagonals starting at $(1,1)$, bypassing elements that are not in lowest terms (e.g., skip $\frac{2}{2}$). This systematic path maps every unique rational number to a distinct natural number position, ensuring completeness.
    
    - _Source_: Understanding Analysis (Abbott) / Countability of Sets.
    
- H5: The correlation coefficient is $\rho = \frac{\text{Cov}(X,Y)}{\sigma_X \sigma_Y}$. By the Cauchy-Schwarz Inequality, its absolute boundaries are strictly bounded: $-1 \le \rho \le 1$.
    
    - _Source_: OpenIntro Statistics / Joint Distributions.
    
- H6: If $f(x) \le g(x) \le h(x)$ for all $x$ in an open interval containing $c$ (except possibly at $c$), and $\lim_{x \to c} f(x) = \lim_{x \to c} h(x) = L$, then $\lim_{x \to c} g(x) = L$.
    
    - _Source_: Calculus (Spivak) / Limit Theorems.
    
- H7: PMF: $P(X = k) = (1-p)^{k-1}p$. Support: $k \in \{1, 2, 3, \dots\}$. (Alternatively, counting failures: $P(X=k) = (1-p)^k p$ for $k \in \{0, 1, 2, \dots\}$).
    
    - _Source_: A First Course in Probability (Ross) / Geometric Models.
    
- H8: Let $(x_n)$ be bounded in $[a,b]$. Divide $[a,b]$ into halves. At least one half must contain infinitely many terms of $(x_n)$; choose this subinterval and pick a point $x_{n_1}$. Bisect this subinterval repeatedly, choosing a subinterval with infinitely many terms and a point $x_{n_k}$ with $n_k > n_{k-1}$. The resulting sequence of nested intervals converges to a point $x$ by the Nested Interval Postulate, and the subsequence $(x_{n_k})$ converges to $x$.
    
    - _Source_: Understanding Analysis (Abbott) / Bolzano-Weierstrass Theorem.
    

## Friday

- F1: Let $T: V \to W$, $U: W \to Z$, and $V: Z \to Y$ be linear transformations with ordered bases $\beta, \gamma, \delta, \epsilon$. Then the composition matrix entry mapping is given by $[VUT]_\beta^\epsilon = [V]_\delta^\epsilon [U]_\gamma^\delta [T]_\beta^\gamma$.
    
    - _Source_: Linear Algebra (Friedberg, Insel, Spence) / Transformation Product Composition.
    
- F2: If $P$ and $Q$ are invertible matrices of appropriate sizes, then $\text{rank}(PA) = \text{rank}(A) = \text{rank}(AQ) = \text{rank}(PAQ)$. Invertible multiplication preserves the dimension of the row space and column space.
    
    - _Source_: Linear Algebra and Its Applications (Lay) / Matrix Invariants.
    
- F3: If a matrix $B$ is obtained from a square matrix $A$ by adding a scalar multiple of one row to another row, then $\det(B) = \det(A)$. The determinant remains completely unchanged under this operation.
    
    - _Source_: Introduction to Linear Algebra (Strang) / Determinant Row Operations.
    
- F4: A subset $K \subseteq \mathbb{R}$ is compact if and only if every sequence $(x_n)$ contained within $K$ has a subsequence $(x_{n_k})$ that converges to a limit point $x$ which is also an element of $K$.
    
    - _Source_: Principles of Mathematical Analysis (Rudin) / Sequential Compactness.
    
- F5: Absolute convergence implies convergence. If $\sum \vert{}a_n\vert{}$ converges, then since $0 \le a_n + \vert{}a_n\vert{} \le 2\vert{}a_n\vert{}$, the series $\sum (a_n + \vert{}a_n\vert{})$ converges by the Comparison Test. Subtracting the convergent series $\sum \vert{}a_n\vert{}$ yields that $\sum a_n$ must also converge.
    
    - _Source_: Understanding Analysis (Abbott) / Absolute Convergence Theorem.
    
- F6: If two continuous random variables satisfy $P(X > t) \le P(Y > t)$ for all $t$ (stochastic dominance), then their respective Cumulative Distribution Functions satisfy $F_X(t) \ge F_Y(t)$ for all $t \in \mathbb{R}$.
    
    - _Source_: Probability and Random Processes (Grimmett & Stirzaker) / Stochastic Orders.
    
- F7: $(\impliedby)$ If $W_1 \subseteq W_2$ or $W_2 \subseteq W_1$, then $W_1 \cup W_2$ equals $W_2$ or $W_1$, which are valid subspaces. $(\implies)$ Assume $W_1 \cup W_2$ is a subspace but $W_1 \not\subseteq W_2$ and $W_2 \not\subseteq W_1$. Choose $w_1 \in W_1 \setminus W_2$ and $w_2 \in W_2 \setminus W_1$. Since both are in the union, $w_1 + w_2 \in W_1 \cup W_2$. If $w_1 + w_2 \in W_1$, then $(w_1 + w_2) - w_1 = w_2 \in W_1$, a contradiction. A symmetric contradiction arises if $w_1+w_2 \in W_2$.
    
    - _Source_: Linear Algebra (Friedberg, Insel, Spence) / Subspace Unions.
    
- F8: The nullity of a linear transformation $T: V \to W$ is the dimension of its null space (kernel), defined as $\text{nullity}(T) = \dim(\{v \in V : T(v) = 0\})$. The rank is the dimension of its range (image), defined as $\text{rank}(T) = \dim(\{T(v) : v \in V\})$.
    
    - _Source_: Linear Algebra (Friedberg, Insel, Spence) / Dimension Theorems.
    

## Saturday

- S1: Let $W_1, W_2 \le V$. Since $0 \in W_1$ and $0 \in W_2$, $0 \in W_1 \cap W_2$. If $x, y \in W_1 \cap W_2$, then $x,y \in W_1 \implies x+y \in W_1$, and $x,y \in W_2 \implies x+y \in W_2$, so $x+y \in W_1 \cap W_2$. For any scalar $c$, $cx \in W_1$ and $cx \in W_2$, so $cx \in W_1 \cap W_2$. Thus, it is a valid subspace.
    
    - _Source_: Linear Algebra and Its Applications (Lay) / Subspace Intersections.
    
- S2: The Borel $\sigma$-field $\mathcal{B}$ on the real line $\mathbb{R}$ is the smallest $\sigma$-field containing all the open intervals (or equivalently, all open sets) in $\mathbb{R}$.
    
    - _Source_: Real Analysis: Modern Techniques and Their Applications (Folland) / Measurable Spaces.
    
- S3: A point $x \in A$ is an isolated point of $A$ if there exists a neighborhood $V_\epsilon(x)$ that contains no other points of $A$ besides $x$. In contrast, $x$ is a limit point of $A$ if _every_ neighborhood $V_\epsilon(x)$ contains at least one point of $A$ distinct from $x$.
    
    - _Source_: Understanding Analysis (Abbott) / Limit Point Geometry.
    
- S4: For a non-negative random variable $X$ and any constant $a > 0$, $P(X \ge a) \le \frac{E[X]}{a}$.
    
    - _Source_: Probability and Computing (Mitzenmacher & Upfal) / Concentration Inequalities.
    
- S5: The zero polynomial is in $P_n(F)$. If $p(x), q(x) \in P_n(F)$, $\deg(p+q) \le \max(\deg p, \deg q) \le n$, so $p+q \in P_n(F)$. For any scalar $c$, $\deg(c \cdot p) \le \deg p \le n$, so $c \cdot p \in P_n(F)$. Since it is closed under addition and scalar multiplication, it is a valid subspace of $P(F)$.
    
    - _Source_: Linear Algebra (Friedberg, Insel, Spence) / Polynomial Subspaces.
    
- S6: Assume $P(A \cap B) = P(A)P(B)$. Then $P(A^c \cap B^c) = P((A \cup B)^c) = 1 - P(A \cup B) = 1 - [P(A) + P(B) - P(A \cap B)] = 1 - P(A) - P(B) + P(A)P(B) = (1 - P(A))(1 - P(B)) = P(A^c)P(B^c)$. This matches the definition of independence.
    
    - _Source_: A First Course in Probability (Ross) / Independence Rules.
    
- S7: For any matrix $A$ over $F$, the row space and column space have the exact same dimension, which is defined as the rank of the matrix ($\dim(\text{Row}(A)) = \dim(\text{Col}(A))$).
    
    - _Source_: Introduction to Linear Algebra (Strang) / Fundamental Subspaces.
    
- S8: Let $X = \sum_{i=1}^r Y_i$, where each $Y_i$ is an independent Geometric random variable with parameter $p$ representing the number of trials to get a single success. By the linearity of expectation, $E[X] = \sum_{i=1}^r E[Y_i] = \sum_{i=1}^r \frac{1}{p} = \frac{r}{p}$.
    
    - _Source_: Mathematical Statistics with Applications (Wackerly) / Negative Binomial Moments.
    

## Sunday

- U1: If an $n \times n$ matrix $A$ contains an entire row of zeros, then $\det(A) = 0$. This can be shown by expanding along that specific zero row.
    
    - _Source_: Advanced Linear Algebra (Roman) / Properties of Determinants.
    
- U2: A sequence $(x_n)$ is a Cauchy sequence if for every $\epsilon > 0$, there exists an $N \in \mathbb{N}$ such that for all $m, n \ge N$, $\vert{}x_n - x_m\vert{} < \epsilon$.
    
    - _Source_: Understanding Analysis (Abbott) / Cauchy Criterion.
    
- U3: Every elementary matrix is invertible. The inverse of an elementary matrix is an elementary matrix of the exact same type, which geometrically reverses the row operation performed (scaling by $1/c$, swapping back, or subtracting the added multiple).
    
    - _Source_: Linear Algebra (Friedberg, Insel, Spence) / Elementary Matrix Inverses.
    
- U4: An infinite series $\sum a_n$ can be rearranged into a new series $\sum a_{\sigma(n)}$ without changing its sum if and only if the series is absolutely convergent ($\sum \vert{}a_n\vert{} < \infty$).
    
    - _Source_: Principles of Mathematical Analysis (Rudin) / Riemann's Rearrangement Theorem.
