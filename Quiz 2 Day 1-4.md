## 🔑 Comprehensive Answer Key

## Day 1 Solutions

- Solution 1.1: Proved. An element $x \in A$ is an interior point if there exists an $\epsilon$-neighborhood $V_\epsilon(x) \subset A$. Consider $1 \in A$. Any neighborhood $(1-\epsilon, 1+\epsilon)$ contains real numbers that are not fractions of the form $\frac{1}{n}$ (for example, choose an irrational number slightly less than $1$). Thus, $1$ is not an interior point, so $A$ cannot be open.
- Solution 1.2: Compute the transformation on the standard basis vectors: $T(1,0,0) = (1, 0)$, $T(0,1,0) = (1, 1)$, and $T(0,0,1) = (0, -1)$. Placing these outputs as columns gives the transformation matrix $\begin{pmatrix} 1 & 1 & 0 \\ 0 & 1 & -1 \end{pmatrix}$.
- Solution 1.3: The product derivative rule gives $f'(x) = e^{-2x} - 2xe^{-2x} = e^{-2x}(1 - 2x)$. Setting $f'(x) = 0$ yields a critical point at $x = \frac{1}{2}$. Since $f'(x) > 0$ for $0 < x < \frac{1}{2}$ and $f'(x) < 0$ for $x > \frac{1}{2}$, the coordinates $( \frac{1}{2}, \frac{1}{2e} )$ locate a local and absolute maximum. There are no local minima.
- Solution 1.4: The marginal distributions are $f_X(x) = \int_x^1 2 \, dy = 2(1-x)$ and $f_Y(y) = \int_0^y 2 \, dx = 2y$. Calculating expectations yields $E[X] = \int_0^1 2x(1-x) \, dx = \frac{1}{3}$, $E[Y] = \int_0^1 2y^2 \, dy = \frac{2}{3}$, and $E[XY] = \int_0^1 \int_x^1 2xy \, dy \, dx = \int_0^1 x(1-x^2) \, dx = \frac{1}{4}$. Thus, $\text{Cov}(X,Y) = E[XY] - E[X]E[Y] = \frac{1}{4} - (\frac{1}{3})(\frac{2}{3}) = \frac{1}{36}$.
- Solution 1.5: $(u + v) + w = u + (v + w)$ for all elements in the space.
- Solution 1.6: A sequence where $x_n > x_{n+1}$ holds true for all indices $n \in \mathbb{N}$.
- Solution 1.7: A collection of pairwise disjoint, exhaustive events whose union equals the complete sample space.
- Solution 1.8: A point $x \in A$ that possesses an open neighborhood containing zero other elements of $A$.
- Solution 1.9: A linear system matching the matrix structure $A\mathbf{x} = \mathbf{0}$.
- Solution 1.10: If $f$ is continuous on $[a,b]$, there exists a point $c \in (a,b)$ such that $\int_a^b f(x) \, dx = f(c)(b-a)$.
- Solution 1.11: The explicit set of points where the probability mass function satisfies $p(x) > 0$.
- Solution 1.12: A real number $b$ such that $a \le b$ for all elements $a$ in the set.
- Solution 1.13: When the column dimension of the left matrix matches the row dimension of the right matrix.
- Solution 1.14: A sequence that is either entirely non-decreasing or entirely non-increasing.

## Day 3 Solutions

- Solution 3.1: Proved. (i) Since $W_1, W_2$ are subspaces, $\mathbf{0} \in W_1$ and $\mathbf{0} \in W_2 \implies \mathbf{0} \in W_1 \cap W_2$. (ii) Let $u, v \in W_1 \cap W_2$. Then $u, v \in W_1 \implies u + v \in W_1$ by closure, and tracking through $W_2$ means $u + v \in W_2 \implies u + v \in W_1 \cap W_2$. (iii) Scaling preserves both properties: $cu \in W_1$ and $cu \in W_2 \implies cu \in W_1 \cap W_2$.
- Solution 3.2: The median $m$ satisfies $\int_0^m e^{-x} \, dx = 0.5 \implies 1 - e^{-m} = 0.5 \implies e^{-m} = 0.5 \implies m = \ln(2)$.
- Solution 3.3: By the Monotone Convergence Theorem, the sequence approaches a limit $L$. Taking the limit on both sides of the recursive definition gives $L = \sqrt{L + 2} \implies L^2 - L - 2 = 0 \implies (L - 2)(L + 1) = 0$. Since the sequence terms are strictly positive, the limit must be $L = 2$.

## Day 4 Solutions

- Solution 4.1: Compute probabilities: $P(X \ge 1) = 1 - P(X = 0) = 1 - (\frac{1}{2})^4 = \frac{15}{16}$. The upper targets are $P(X = 3) = \binom{4}{3}(\frac{1}{2})^4 = \frac{4}{16}$ and $P(X = 4) = \binom{4}{4}(\frac{1}{2})^4 = \frac{1}{16}$, so $P(X \ge 3) = \frac{5}{16}$. By conditional probability definitions: $\frac{5/16}{15/16} = \frac{1}{3}$.
- Solution 4.2: Expanding along row 1: $\det(A) = 2 \cdot (1 \cdot (-1) - 0 \cdot 2) - 0 + 1 \cdot (3 \cdot 2 - 1 \cdot 0) = -2 + 6 = 4$. Since $\det(A) \neq 0$, the matrix is completely invertible.
- Solution 4.3: The intersection contains only the overlapping single point boundary set $A \cap B = \{1\}$. The union is $A \cup B = [0, 2)$, so its complement maps all exterior positions: $(A \cup B)^c = (-\infty, 0) \cup [2, \infty)$.
- Solution 4.4: Find the second derivative: $f'(x) = 2x \ln(x) + x$, and $f''(x) = 2 \ln(x) + 2 + 1 = 2 \ln(x) + 3$. Setting $f''(x) > 0 \implies 2 \ln(x) > -3 \implies \ln(x) > -1.5 \implies x > e^{-1.5}$. Thus, the function is concave upward on the interval $(e^{-1.5}, \infty)$.
- Solution 4.5: For any matrix $A$, $\text{rank}(A) + \text{nullity}(A) = n$, where $n$ is the column dimension count.
- Solution 4.6: A real number that acts as a lower bound for a set and is greater than or equal to any other lower bound for that same set.
- Solution 4.7: Events where the joint probability matches the product of their individual probabilities: $P(A \cap B) = P(A)P(B)$.
- Solution 4.8: The unique vector of scalar coefficients required to write a vector as a linear combination of a given ordered basis.
- Solution 4.9: A property where a single choice of $\delta > 0$ satisfies continuity bounds across the entire set domain simultaneously.
