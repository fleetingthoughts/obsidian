## 📅 MONDAY: MIXED SPRINT 1 – ANSWER KEY

## Q1. Exponential Definitions of $\sinh x$ and $\cosh x$

- Full Answer:  
    $$\sinh x = \frac{e^x - e^{-x}}{2}, \quad \cosh x = \frac{e^x + e^{-x}}{2}$$
- Theorems/Definitions Leveraged: Definition of Hyperbolic Functions (Section 3.5a) (p. 12).
- Where It Comes From / How It Was Proved: Derived by splitting the general exponential function $e^x$ into its even and odd component parts via the identity $e^x = \frac{e^x + e^{-x}}{2} + \frac{e^x - e^{-x}}{2}$. By definition, the symmetric (even) part is named $\cosh x$ and the asymmetric (odd) part is named $\sinh x$.
- Main Point & Significance: Tests basic structural recognition. Hyperbolic functions form the foundational parameterization of the hyperbola ($x^2 - y^2 = 1$), mirroring how circular functions ($\cos\theta, \sin\theta$) parameterize the unit circle ($x^2 + y^2 = 1$) (p. 13).

## Q2. Formal Asymptotic Definition of $f(x) = O(g(x))$

- Full Answer: $f(x) = O(g(x))$ as $x \to \infty$ if there exist positive constants $C$ and $x_0$ such that:  
    $$\vert f(x) \vert \leq C \vert g(x) \vert \quad \text{for all } x \geq x_0$$
- Theorems/Definitions Leveraged: Big-$O$ Order of Magnitude notation (Section 3.7f) (p. 12).
- Where It Comes From / How It Was Proved: Paul Bachmann (1894) and Edmund Landau introduced this notation. It is verified by computing the upper bound of the absolute ratio $\frac{\vert{}f(x)\vert{}}{\vert{}g(x)\vert{}}$ as $x \to \infty$ and demonstrating it remains bounded by a finite real number $C$.
- Main Point & Significance: Eliminates minor local algebraic noise to clarify dominant long-term trends. Essential for verifying algorithm complexity, convergence rates of infinite series, and numerical stability (pp. 12, 18).

## Q3. Prerequisite Condition for Rational Integrands $\int \frac{P(x)}{Q(x)}\,dx$

- Full Answer: Polynomial long division must be executed immediately to express the improper fraction as a proper fraction:  
    $$\frac{P(x)}{Q(x)} = S(x) + \frac{R(x)}{Q(x)}$$  
    where $\text{deg}(R) < \text{deg}(Q)$, and $S(x)$ is a polynomial quotient.
- Theorems/Definitions Leveraged: Integration of Rational Functions (Section 3.12) (p. 13).
- Where It Comes From / How It Was Proved: Follows directly from the Euclidean Division Theorem for polynomial rings. If the degree of the numerator matches or exceeds the denominator, the fraction represents an unreduced improper form that cannot be resolved via partial fraction expansion decomposition.
- Main Point & Significance: Tests procedural discipline. Skipping long division renders partial fraction expansion algebraically impossible, because the underlying linear systems assume a strictly proper fractional input (p. 13).

## Q4. Evaluate: $\int x^2 e^{3x} \, dx$

- Full Answer: Use Integration by Parts twice. Let $u = x^2 \implies du = 2x\,dx$, and $dv = e^{3x}\,dx \implies v = \frac{1}{3}e^{3x}$.  
    $$\int x^2 e^{3x} \, dx = \frac{1}{3}x^2 e^{3x} - \frac{2}{3}\int x e^{3x} \, dx$$  
    Apply parts again to the remaining integral: Let $u = x \implies du = dx$, and $dv = e^{3x}\,dx \implies v = \frac{1}{3}e^{3x}$.  
    $$\int x e^{3x} \, dx = \frac{1}{3}x e^{3x} - \frac{1}{3}\int e^{3x}\,dx = \frac{1}{3}x e^{3x} - \frac{1}{9}e^{3x}$$  
    Substitute back and distribute coefficients:  
    $$\int x^2 e^{3x} \, dx = \frac{1}{3}x^2 e^{3x} - \frac{2}{9}x e^{3x} + \frac{2}{27}e^{3x} + C = e^{3x}\left(\frac{x^2}{3} - \frac{2x}{9} + \frac{2}{27}\right) + C$$
- Theorems/Definitions Leveraged: Integration by Parts (Section 3.11a) (p. 13).
- Where It Comes From / How It Was Proved: Derived directly by integrating the product rule of differentiation: $\frac{d}{dx}(uv) = u\frac{dv}{dx} + v\frac{du}{dx}$. Integrating both sides fields $uv = \int u\,dv + \int v\,du$, which rearranges directly into the theorem.
- Main Point & Significance: Tests iterative operator reduction. It measures your capacity to sequentially step down the algebraic degree of polynomial components when paired against an invariant exponential operator (p. 13).

## Q5. Differentiate: $f(x) = \ln(\cosh x)$

- Full Answer: Apply the Chain Rule. Let $u = \cosh x$, so $f(u) = \ln u$.  
    $$\frac{df}{dx} = \frac{1}{\cosh x} \cdot \frac{d}{dx}(\cosh x) = \frac{1}{\cosh x} \cdot \sinh x = \tanh x$$
- Theorems/Definitions Leveraged: Differentiation of Hyperbolic Functions (Section 3.5b) (p. 12).
- Where It Comes From / How It Was Proved: Proved by expressing the derivative via exponentials: $\frac{d}{dx}(\cosh x) = \frac{d}{dx}\left(\frac{e^x + e^{-x}}{2}\right) = \frac{e^x - e^{-x}}{2} = \sinh x$.
- Main Point & Significance: Reinforces structural tracking across trans-trigonometric groups. It highlights the clean structural parallel to circular calculus, where $\frac{d}{dx}(\ln(\cos x)) = -\tan x$, noting the absence of the negative sign due to hyperbolic identity variations (p. 12).

## Q6. Evaluate: $\int \frac{1}{x^2 - 5x + 6} \, dx$

- Full Answer: Factor the quadratic denominator: $x^2 - 5x + 6 = (x-2)(x-3)$. Apply a partial fraction split:  
    $$\frac{1}{(x-2)(x-3)} = \frac{A}{x-2} + \frac{B}{x-3} \implies 1 = A(x-3) + B(x-2)$$
    
    - Set $x = 3 \implies 1 = B(1) \implies B = 1$.
    - Set $x = 2 \implies 1 = A(-1) \implies A = -1$.  
        $$\int \left( \frac{1}{x-3} - \frac{1}{x-2} \right) \,dx = \ln\vert{}x-3\vert{} - \ln\vert{}x-2\vert{} + C = \ln\left\vert{} \frac{x-3}{x-2} \right\vert{} + C$$
    
- Theorems/Definitions Leveraged: Method of Undetermined Coefficients (Section 3.12d) (p. 13).
- Where It Comes From / How It Was Proved: Rooted in the Fundamental Theorem of Algebra, which asserts every real polynomial can be completely resolved into linear and irreducible quadratic factors over $\mathbb{R}$, guaranteeing a unique decomposition template (p. 13).
- Main Point & Significance: Tests mapping of rational fields to logarithmic fields. Translates complex algebraic quotients into basic linear sum expressions (p. 13).

## Q7. Evaluate: $\int \frac{1}{1 + \sin x} \, dx$

- Full Answer: Apply the universal Weierstrass substitution: $t = \tan(x/2)$, where $\sin x = \frac{2t}{1+t^2}$ and $dx = \frac{2}{1+t^2}\,dt$ (pp. 13, 36).  
    $$\int \frac{1}{1 + \frac{2t}{1+t^2}} \cdot \frac{2}{1+t^2}\,dt = \int \frac{2}{1+t^2 + 2t}\,dt = \int \frac{2}{(t+1)^2}\,dt = -\frac{2}{t+1} + C$$  
    Reverting to our original variable $x$:  
    $$\text{Result} = -\frac{2}{\tan\left(\frac{x}{2}\right) + 1} + C$$
- Theorems/Definitions Leveraged: Rational Representation of Trigonometric Curves (Section 3.13b) (p. 13).
- Where It Comes From / How It Was Proved: Geometrically derived by projecting points on the unit circle from the vertex $(-1,0)$ onto the vertical line $x=1$. This map algebraically rationalizes the transcendental curve of the circle (p. 13).
- Main Point & Significance: Demonstrates a universal bridge between transcendental and algebraic structures. Any rational combination of sine and cosine can be completely rationalized through this single map (p. 13).

## Q4. Prove Asymptotic Small-$o$ Condition: $x^2 \ln x = o(x^{2.01})$

- Theorems/Definitions Leveraged: Small-$o$ Relative Growth Definition (Section 3.7f) (p. 12).
- Where It Comes From / How It Was Proved: Rooted in the property that any positive power of $x$, no matter how tiny ($\alpha = 0.01$), will eventually outgrow and dominate a logarithmic function ($\ln x$) as $x \to \infty$ (p. 12).

## Q6. Evaluate: $\int \frac{2x + 5}{x^2 + 4x + 5} \, dx$

## Q7. Compute the Exact Boundary Derivative: $\frac{d}{dx} \int_{x}^{x^2} \ln(t^2 + 1) \, dt$

- Full Answer: Apply the Fundamental Theorem of Calculus paired with the Chain Rule (Leibniz Integral Rule):  
    $$\frac{d}{dx} \int_{g(x)}^{h(x)} f(t)\,dt = f(h(x))\cdot h'(x) - f(g(x))\cdot g'(x)$$  
    Substitute $h(x) = x^2 \implies h'(x) = 2x$, and $g(x) = x \implies g'(x) = 1$:  
    $$\text{Result} = \ln((x^2)^2 + 1) \cdot (2x) - \ln(x^2 + 1) \cdot (1) = 2x \ln(x^4 + 1) - \ln(x^2 + 1)$$
- Theorems/Definitions Leveraged: The Derivative of the Integral (Section 2.9a / Leibniz Rule) (p. 11).
- Where It Comes From / How It Was Proved: If $F'(t) = f(t)$, then $\int_{g(x)}^{h(x)} f(t)\,dt = F(h(x)) - F(g(x))$. Differentiating this expression with respect to $x$ via the chain rule fields $F'(h(x))h'(x) - F'(g(x))g'(x)$, which evaluates directly to $f(h(x))h'(x) - f(g(x))g'(x)$ (p. 11).
- Main Point & Significance: Evaluates variable tracking across active boundaries. Avoids the need for explicit integration by treating the integral operator as a direct inverse mapping function (p. 11).
