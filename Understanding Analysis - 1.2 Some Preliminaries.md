# Induction
The fundamental principle behind induction is that if $S$ is some subset of $\mathbb{N}$ with the property:
1) $S$ contains 1
2) If $S$ contains $n$, then it contains $n+1$
Then $S = \mathbb{N}$ . 
Why is it a valid argumentative technique and what are its implications on natural numbers? This will be discussed later.
