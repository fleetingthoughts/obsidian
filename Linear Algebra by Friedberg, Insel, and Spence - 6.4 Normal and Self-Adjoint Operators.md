# Definitions

***Definition (normal transformation and normal matrix).*** Let $V$ be an inner product space and $T$ be a linear operator. We say $T$ is normal if $TT^*=T^*T$. Similarly, a matrix $A$ (i.e. the linear transformation $L_A$) is normal if $A^*A=AA^*$

***Definition (Hermitian matrix or self-adjoint matrix).*** A matrix $A$ is self-adjoint (or Hermitian) if $A=A^*$.
