## 🔑 Answers, Blueprints, & Book Mapping

## 🔹 Day 1 Solutions

- 1.1 Answer: (1) The zero vector is in $W$; (2) $x+y \in W$ whenever $x,y \in W$; (3) $cx \in W$ whenever $c \in F$ and $x \in W$. (Friedberg, Ch 1.3 - Subspaces)
- 1.2 Answer: $P(A \cap B) = P(A) \cdot P(B)$. (Hogg, Ch 1.4 - Independence)
- 1.3 Answer: A sequence $\{x_n\}$ where for every $\epsilon > 0$, there exists an index $N \in \mathbb{N}$ such that for all $n, m \geq N$, $|x_n - x_m| < \epsilon$. (Abbott, Ch 2.6 - Cauchy Criterion)
- 1.4 Answer: The Mean Value Theorem: $f'(c) = \frac{f(b) - f(a)}{b - a}$. (Courant, Ch 2.8 - Mean Value Theorem)
- 1.5 Proof Blueprint: Let $N(T) = \{v \in V : T(v) = 0\}$. (1) $T(0) = 0 \implies 0 \in N(T)$. (2) Let $x, y \in N(T) \implies T(x+y) = T(x) + T(y) = 0 + 0 = 0 \implies x+y \in N(T)$. (3) Let $c \in F, x \in N(T) \implies T(cx) = cT(x) = c(0) = 0 \implies cx \in N(T)$. Valid subspace. (Friedberg, Ch 2.1 - Linear Transformations)

## 🔹 Day 2 Solutions

- 2.1 Answer: Every non-empty set of real numbers that is bounded above has a least upper bound (supremum) in $\mathbb{R}$. (Abbott, Ch 1.3 - Axiom of Completeness)
- 2.2 Answer: If $\{A_n\}$ is a sequence of mutually exclusive events, then $P(\bigcup_{n=1}^{\infty} A_n) = \sum_{n=1}^{\infty} P(A_n)$. (Hogg, Ch 1.3 - Probability Set Function)
- 2.3 Answer: A linearly independent subset of the space that spans the entire space. (Friedberg, Ch 1.6 - Bases and Dimension)
- 2.4 Answer: The sequence is guaranteed to converge to a finite limit. (Monotone Convergence Theorem). (Abbott, Ch 2.4 / Courant, Ch 1.7 - Monotone Sequences)
- 2.5 Proof Blueprint: Note that $\mathcal{C} = \mathcal{C} \cup \emptyset$, and $\mathcal{C} \cap \emptyset = \emptyset$. By the additivity axiom, $P(\mathcal{C}) = P(\mathcal{C}) + P(\emptyset)$. Since $P(\mathcal{C}) = 1$ is finite, subtract $P(\mathcal{C})$ from both sides to get $1 = 1 + P(\emptyset) \implies P(\emptyset) = 0$. (Hogg, Ch 1.3 - Theorem 1.3.2)

## 🔹 Day 3 Solutions

- 3.1 Answer: The Dimension Theorem (or Rank-Nullity Theorem): $\dim(V) = \text{rank}(T) + \text{nullity}(T)$. (Friedberg, Ch 2.1 - Transformations)
- 3.2 Answer: Closed intervals contain their boundary endpoints ($a \leq x \leq b$); open intervals strictly exclude their boundary endpoints ($a < x < b$). (Courant, Ch 1.1 / Abbott, Ch 1.3 - Neighborhoods)
- 3.3 Answer: The mean ($\mu$) and the variance ($\sigma^2$). (Hogg, Ch 3.4 - The Normal Distribution)
- 3.4 Answer: The number of columns in $A$ must equal the number of rows in $B$. (Friedberg, Ch 2.3 - Matrix Multiplication)
- 3.5 Proof Blueprint: Assume for contradiction that there exists an $\epsilon_0 > 0$ such that no $s \in S$ satisfies $u - \epsilon_0 < s$. This implies that for all $s \in S$, $s \leq u - \epsilon_0$. This makes $u - \epsilon_0$ an upper bound for $S$. However, $u - \epsilon_0 < u$, which contradicts the definition of $u$ as the _least_ upper bound. Thus, the original claim must hold. (Abbott, Ch 1.3 - Lemma 1.3.8)

## 🔹 Day 4 Solutions

- 4.1 Answer: The derivative of the integral accumulation function is the integrand itself: $G'(x) = f(x)$. (Courant, Ch 2.9 - Fundamental Theorem of Calculus)
- 4.2 Answer: A set $A$ is countable if it is finite or if it has the same cardinality as the natural numbers $\mathbb{N}$ (i.e., $\mathbb{N} \sim A$). (Abbott, Ch 1.5 - Cardinality)
- 4.3 Answer: $P(B|A) = \frac{P(A \cap B)}{P(A)}$, provided that $P(A) > 0$. (Hogg, Ch 1.4 - Conditional Probability)
- 4.4 Answer: An expression of the form $c_1 v_1 + c_2 v_2 + \dots + c_n v_n$, where $c_i$ are scalars from the field $F$. (Friedberg, Ch 1.4 - Linear Combinations)
- 4.5 Proof Blueprint: Let $\epsilon > 0$. Since $x_n \to X$, $\exists N_1$ such that $\forall n \geq N_1, |x_n - X| < \frac{\epsilon}{2}$. Since $y_n \to Y$, $\exists N_2$ such that $\forall n \geq N_2, |y_n - Y| < \frac{\epsilon}{2}$. Choose $N = \max(N_1, N_2)$. For all $n \geq N$, $|(x_n + y_n) - (X + Y)| \leq |x_n - X| + |y_n - Y| < \frac{\epsilon}{2} + \frac{\epsilon}{2} = \epsilon$. Thus, $x_n + y_n \to X + Y$. (Abbott, Ch 2.3 - Order Limit Theorems)

## 🔹 Day 5 Solutions

- 5.1 Answer: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$. (Hogg, Ch 1.3 - Theorem 1.3.5)
- 5.2 Answer: Every bounded sequence of real numbers contains a convergent subsequence. (Abbott, Ch 2.5 - Subsequences)
- 5.3 Answer: (1) $T(x+y) = T(x) + T(y)$ for all $x,y \in V$; (2) $T(cx) = cT(x)$ for all $c \in F, x \in V$. (Friedberg, Ch 2.1 - Linear Transformations)
- 5.4 Answer: The sequence of partial sums $S_k = \sum_{n=1}^{k} a_n$ must converge to a finite, definite limit as $k \to \infty$. (Courant, Ch 1.7 - Infinite Series)
- 5.5 Proof Blueprint: ($\implies$) Let $v = w_1 + w_2 = w_1' + w_2'$. Rearranging gives $w_1 - w_1' = w_2' - w_2$. The left side is in $W_1$, right side is in $W_2$. Thus, the element is in $W_1 \cap W_2 = \{0\} \implies w_1 = w_1'$ and $w_2 = w_2'$ (Uniqueness). ($\impliedby$) Let $x \in W_1 \cap W_2$. We can express the zero vector as $0 = x + (-x) = 0 + 0$. Since representations must be unique, $x = 0$. Thus, $W_1 \cap W_2 = \{0\}$. (Friedberg, Ch 1.3 - Exercise 30)

## 🔹 Day 6 Solutions

- 6.1 Answer: $E[X] = \int_{-\infty}^{\infty} x \cdot f(x) \, dx$. (Hogg, Ch 1.8 - Expectation)
- 6.2 Answer: Given any real number $x \in \mathbb{R}$, there exists a natural number $n \in \mathbb{N}$ such that $n > x$. (Abbott, Ch 1.4 - Archimedean Property)
- 6.3 Answer: $T$ must be both one-to-one (injective) and onto (surjective). (Friedberg, Ch 2.4 - Invertibility)
- 6.4 Answer: The sum of the main diagonal entries of the matrix: $\text{tr}(M) = \sum_{i=1}^{n} M_{ii}$. (Friedberg, Ch 1.3 - Subspaces)
- 6.5 Proof Blueprint: Let $u = \sup(B)$. By definition, $b \leq u$ for all $b \in B$. Since $A \subseteq B$, it follows that $a \leq u$ for all $a \in A$. Therefore, $u$ is an upper bound for the set $A$. Since $\sup(A)$ is the _least_ upper bound of $A$, it must be less than or equal to any other upper bound of $A$, meaning $\sup(A) \leq u = \sup(B)$. (Abbott, Ch 1.3 - Supremum Properties)

## 🔹 Day 7 Solutions

- 7.1 Answer: $f_X(x) = \int_{-\infty}^{\infty} f(x, y) \, dy$. (Hogg, Ch 2.1 - Marginal Distributions)
- 7.2 Answer: Assume $(p/q)^2 = 2$ where $p, q$ are integers with no common factors. This implies $p^2 = 2q^2$, meaning $p^2$ is even, so $p$ is even ($p = 2r$). Substituting yields $4r^2 = 2q^2 \implies q^2 = 2r^2$, meaning $q$ is also even. This contradicts the assumption of no common factors. (Abbott, Ch 1.1 / Courant, Ch 1.1 - Irrationality)
- 7.3 Answer: The only scalar combination that satisfies $c|c_1 v_1 + c_2 v_2 + \dots + c_n v_n = 0$ is the trivial solution where all scalars $c_i = 0$. (Friedberg, Ch 1.5 - Linear Independence)
- 7.4 Answer: The open interval $(x_0 - \epsilon, x_0 + \epsilon)$ centered at $x_0$ with a radius of $\epsilon > 0$. (Courant, Ch 1.1 / Abbott, Ch 1.2 - Neighborhoods)
- 7.5 Proof Blueprint: We need to show $P(A \cap B^c) = P(A)P(B^c)$. Note that $A = (A \cap B) \cup (A \cap B^c)$, which is a disjoint union. Therefore, $P(A) = P(A \cap B) + P(A \cap B^c)$. Since $A$ and $B$ are independent, substitute $P(A \cap B) = P(A)P(B)$. This yields $P(A) = P(A)P(B) + P(A \cap B^c) \implies P(A \cap B^c) = P(A) - P(A)P(B) = P(A)(1 - P(B))$. Since $1 - P(B) = P(B^c)$, we get $P(A \cap B^c) = P(A)P(B^c)$. Independence is proved. (Hogg, Ch 1.4 - Independence Properties)
