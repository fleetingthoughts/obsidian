Abbott provides motivating questions for why we study the special limits that are derivatives:
- Are continuous functions always differentiable? If not, how nondifferentiable can a continuous function be? 
- Are differentiable functions continuous
- If a function has a derivative, what can we say about the derivative itself? 
	- Is the derivative continuous?
	- Are there restrictions we can put to produce the properties we desire such as continuity.

In an interesting pedagogical fashion, Abbott opens with a pathological function and problem as he has done for all the previous sections. We begin with a variant of the topologist's sine curve: $$f(x) = \begin{cases} x^n\sin\left( \frac{1}{x} \right) & \text{if } x \ne 0 \\ 0 & \text{if } x = 0 \end{cases}$$
This variant of the topologist's sine curve has the interesting property where a continuous function may have a noncontinuous $m$-th derivative depending on what we set $n$ to which gives us a mathematical experiment to see what kind of restrictions are required to produce the desired properties in a derivative. 

This pathology will be returned to throughout the chapter.
