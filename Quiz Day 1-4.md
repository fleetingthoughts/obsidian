## 🗝️ Day 1 Answer Key

1. One-to-One and Null Space Proof:
    
    - _Theorem_: $T$ is one-to-one $\iff N(T) = \{0\}$.
    - _Proof ($\implies$)_: Let $x \in N(T)$. Then $T(x) = 0 = T(0)$. Since $T$ is one-to-one, $x = 0$. Hence $N(T) = \{0\}$.
    - _Proof ($\impliedby$)_: Suppose $T(u) = T(v)$. Then $T(u) - T(v) = 0 \implies T(u-v) = 0$. Thus $(u-v) \in N(T)$. Since $N(T) = \{0\}$, $u-v = 0 \implies u = v$. Thus $T$ is one-to-one.
    
2. Matrix Representation Algorithm $[T]_\beta^\gamma$:
    
    - Let $\beta = \{v_1, \dots, v_n\}$ and $\gamma = \{w_1, \dots, w_m\}$.
    - _Step 1_: Compute $T(v_j)$ for each basis vector in $\beta$.
    - _Step 2_: Express each $T(v_j)$ as a unique linear combination of vectors in $\gamma$: $T(v_j) = \sum_{i=1}^m A_{ij}w_i$.
    - _Step 3_: Place the unique coordinates $A_{1j}, \dots, A_{mj}$ into column $j$ of matrix $A$. The resulting $m \times n$ matrix is $[T]_\beta^\gamma$.
    
3. Limit Point of a Sequence vs. Set:
    
    - _Sequence limit point_: A real number $x$ where every $\epsilon$-neighborhood $(x-\epsilon, x+\epsilon)$ contains infinitely many _indices_ $n$ such that $a_n \in (x-\epsilon, x+\epsilon)$.
    - _Set limit point_: A real number $x$ where every $\epsilon$-neighborhood contains at least one point $a \in A$ such that $a \neq x$. A sequence limit point can be a point that repeats infinitely (e.g., $1, 0, 1, 0\dots$ has limit points $\{0, 1\}$), whereas a finite set has no limit points.
    
4. Transformation Composition Matrix:
    
    - $[UT]_\beta^\alpha = [U]_\gamma^\alpha [T]_\beta^\gamma$.
    
5. Countable Unions & Ring Construction Technique:
    
    - Given an arbitrary sequence of events $\{C_n\}_{n=1}^\infty$, they generally overlap. To leverage the countable additivity axiom (which requires disjoint sets), construct mutually exclusive "ring" sets: $D_1 = C_1$, and $D_n = C_n \setminus \left(\bigcup_{i=1}^{n-1} C_i\right)$ for $n > 1$.
    - By construction, $\bigcup_{n=1}^\infty C_n = \bigcup_{n=1}^\infty D_n$ and $D_i \cap D_j = \emptyset$ for $i \neq j$. You can then assert $P(\bigcup C_n) = P(\bigcup D_n) = \sum P(D_n)$.
    
6. Infimum Criteria:
    
    - A set $A \subseteq \mathbb{R}$ is bounded below if $\exists M \in \mathbb{R}$ such that $x \geq M$, $\forall x \in A$.
    - The infimum $\alpha = \inf A$ satisfies: (1) $\alpha$ is a lower bound ($\forall x \in A, x \geq \alpha$). (2) If $\gamma$ is any lower bound of $A$, then $\gamma \leq \alpha$. (Equivalently: $\forall \epsilon > 0$, $\exists x \in A$ with $x < \alpha + \epsilon$).
    
7. Dimension of $\mathcal{L}(V,W)$:
    
    - $\dim(\mathcal{L}(V,W)) = \dim(V) \cdot \dim(W)$.
    
8. Distance Function (Metric) Axioms:
    
    - (1) Positivity/Separation: $d(x, y) \geq 0$, and $d(x, y) = 0 \iff x = y$.
    - (2) Symmetry: $d(x, y) = d(y, x)$.
    - (3) Triangle Inequality: $d(x, z) \leq d(x, y) + d(y, z)$.
    
9. Expectation Linearity Proof:
    
    - $E[aX + bY] = \int_{-\infty}^\infty \int_{-\infty}^\infty (ax + by) f(x, y) \,dx\,dy = a \int_{-\infty}^\infty x \left[ \int_{-\infty}^\infty f(x, y) \,dy \right] dx + b \int_{-\infty}^\infty y \left[ \int_{-\infty}^\infty f(x, y) \,dx \right] dy$.
    - Since the inner brackets yield marginal densities $f_X(x)$ and $f_Y(y)$, this evaluates exactly to $a \int x f_X(x)\,dx + b \int y f_Y(y)\,dy = aE[X] + bE[Y]$.
    
10. Uniqueness of $[T]_\beta^\gamma$:
    
    - Because any vector in $W$ is uniquely expressible as a linear combination of vectors in the basis $\gamma$, the coefficients $A_{ij}$ mapping $T(v_j) = \sum A_{ij}w_i$ are uniquely determined for each column $j$.
    
11. De Morgan's Laws (Countable):
    
    - $\left(\bigcup_{n=1}^\infty A_n\right)^c = \bigcap_{n=1}^\infty A_n^c$ and $\left(\bigcap_{n=1}^\infty A_n\right)^c = \bigcup_{n=1}^\infty A_n^c$.
    
12. Coordinate Vector Identity:

13. Kolmogorov Axioms:

    - (2) $P(\mathcal{C}) = 1$.
    - (3) Countable Additivity: If $A_m \cap A_n = \emptyset$ for all $m \neq n$, then $P(\bigcup_{n=1}^\infty A_n) = \sum_{n=1}^\infty P(A_n)$.

By the replacement theorem, we can sequentially replace all $n$ vectors of $\beta$ with vectors from $S$ while maintaining a spanning set. Since $m > n$, there will be vectors remaining in $S$ that have not been used. These remaining vectors must be linear combinations of the newly replaced vectors in the span, directly violating the linear independence of $S$.

At least one half must contain infinite elements; designate it $I_1$. Inductively repeat this step to create a sequence of nested closed intervals $I_n$ whose lengths $\to 0$. By the Nested Interval Postulate, their intersection defines a unique real number, which serves as the targeted limit point.

12. Law of Total Probability:

    - Suppose $\mathbb{N}$ is bounded above. By the Axiom of Completeness, it has a least upper bound $\alpha = \sup \mathbb{N}$. Then $\alpha - 1$ cannot be an upper bound, meaning $\exists n \in \mathbb{N}$ such that $n > \alpha - 1$. Adding 1 yields $n + 1 > \alpha$. Since $n+1 \in \mathbb{N}$, this contradicts that $\alpha$ is an upper bound.
    
15. Row Interchange Determinant:
    
    - Interchanging any two rows of a matrix multiplies the determinant by $-1$.
    
16. Matrix Transpose Invertibility Proof:
    
    - If $A$ is invertible, $AA^{-1} = I \implies (AA^{-1})^t = I^t \implies (A^{-1})^t A^t = I$. Similarly, $A^{-1}A = I \implies A^t (A^{-1})^t = I$. Thus, $A^t$ is invertible, and its inverse is given by $(A^t)^{-1} = (A^{-1})^t$.

The resulting right side matrix is $A^{-1}$, yielding $(I \mid A^{-1})$. This works because each elementary row operation corresponds to left-multiplying by an elementary matrix $E_k$, meaning $E_k \dots E_1 (A \mid I) = (IA \mid I) = (I \mid A^{-1})$ since $A^{-1} = E_k \dots E_1$.

## 🗝️ Day 3 Answer Key

1. Row Rank, Column Rank, and Equivalence:
    
    - Row rank is the dimension of the subspace spanned by the rows; column rank is the dimension of the subspace spanned by the columns.
    - _Theorem_: For any matrix over a field $F$, the row rank always equals the column rank (this shared value is defined as $\text{rank}(A)$).
    
2. Cauchy Convergence Proof Framework:
    
    - To show convergence using the Cauchy Criterion without computing the explicit limit, bounded variation or structural absolute constraints must be bounded sequentially. Show that for indices far out, the trailing tail terms shrink below $\epsilon$ via a geometric or dominating series control, proving the sequence is Cauchy. Since $\mathbb{R}$ is complete, any Cauchy sequence automatically converges to a real limit.
    
3. Perfect Set:
    
    - A set $P \subseteq \mathbb{R}$ that is closed and contains no isolated points (meaning every point in $P$ is a limit point of $P$).
    
4. Rank of a Matrix Product:
    
    - $\text{rank}(AB) \leq \min(\text{rank}(A), \text{rank}(B))$.
    
5. Moment Calculation via MGF Algorithm:
    
    - Take the $n$-th derivative of the MGF with respect to $t$, and evaluate it at $t=0$: $E[X^n] = \left. \frac{d^n}{dt^n} M(t) \right\vert{}_{t=0}$.
    
6. Moment-Generating Function Definition:
    
    - $M(t) = E[e^{tX}] = \int_{-\infty}^\infty e^{tx}f(x)\,dx$. It must exist and be finite for all $t$ in an open interval $(-h, h)$ for some $h > 0$.
    
7. Rank of Transpose Theorem:
    
    - $\text{rank}(A) = \text{rank}(A^t)$.
    
8. Discrete Conditional Distribution:
    
    - $p_{X\mid Y}(x \mid y) = P(X = x \mid Y = y) = \frac{p(x, y)}{p_Y(y)}$, provided $p_Y(y) > 0$.
    
9. Multiplicativity of Determinants Proof:
    
    - Case 1: If $\text{rank}(A) < n$, then $\det(A) = 0$. Since $\text{rank}(AB) \leq \text{rank}(A) < n$, $\det(AB) = 0$. Thus $\det(AB) = 0 = 0 \cdot \det(B) = \det(A)\det(B)$.
    - Case 2: If $\text{rank}(A) = n$, then $A$ is invertible and can be written as a product of elementary matrices: $A = E_k \dots E_1$. Since left-multiplying by an elementary matrix performs a row operation, show $\det(EB) = \det(E)\det(B)$ holds for each base type. Inductively, $\det(AB) = \det(E_k \dots E_1 B) = \det(E_k) \dots \det(E_1)\det(B) = \det(E_k \dots E_1)\det(B) = \det(A)\det(B)$.
    
10. Squeeze Theorem:
    
    - If sequences satisfy $a_n \leq b_n \leq c_n$ for all $n \geq N$, and $\lim a_n = \lim c_n = L外部$, then $\lim b_n = L$.
    
11. Multi-Linearity of Rows:
    
    - The determinant is a linear function of each individual row vector when all remaining $n-1$ row vectors are held fixed.
    
12. Compact Set Cover Criterion:
    
    - A set $K \subseteq \mathbb{R}$ is compact if every open cover of $K$ has a finite subcover.
    
13. Nonhomogeneous System Solution Structure:
    
    - _Verification_: Check consistency by confirming $\text{rank}(A) = \text{rank}(A \mid b)$.
    - _Construction_: Find a single particular solution $x_p$ such that $A x_p = b$. The complete general solution set is formulated as $K = \{x_p + x_h : x_h \in N(A)\}$, where $N(A)$ is the solution space of the homogeneous system.
    
14. Matrix Product Composition Theorem:
    
    - Let $T: V \to W$ and $U: W \to Z$. Under ordered bases $\alpha, \beta, \gamma$, the entry-wise composition matrix matches matrix multiplication: $[UT]_\alpha^\gamma = [U]_\beta^\gamma [T]_\alpha^\beta$.
    
15. Bolzano-Weierstrass Proof (Bisection Strategy):
    
    - Since the sequence is bounded, it is contained in a closed interval $I_0 = [a, b]$. Bisect $I_0$ into two halves. At least one half must contain infinitely many terms of the sequence; choose this half and call it $I_1$. Continue bisecting inductively to create nested closed intervals $I_n$ with length $(b-a)/2^n \to 0$. By the Nested Interval Postulate, there exists a unique point $x$ contained in all $I_n$. Construct a subsequence by picking $a_{n_k} \in I_k$ such that $n_k > n_{k-1}$. Since the interval lengths tend to 0, this subsequence converges to $x$.
    
16. Absolute Convergence Test:
    
    - Absolute convergence implies convergence. If $\sum \vert{}a_n\vert{}$ converges, then for $\epsilon > 0$, $\exists N$ such that $\forall m > n \geq N$, $\sum_{i=n+1}^m \vert{}a_i\vert{} < \epsilon$. By the triangle inequality, $\left\vert{}\sum_{i=n+1}^m a_i\right\vert{} \leq \sum_{i=n+1}^m \vert{}a_i\vert{} < \epsilon$, which proves that the partial sums of $\sum a_n$ satisfy the Cauchy Criterion and therefore converge.
    
17. Poisson MGF Derivation:
    
    - $M(t) = E[e^{tX}] = \sum_{x=0}^\infty e^{tx} \frac{\lambda^x e^{-\lambda}}{x!} = e^{-\lambda} \sum_{x=0}^\infty \frac{(\lambda e^t)^x}{x!}$.
    - Recognizing the infinite series as the Taylor expansion of $e^{\lambda e^t}$, this evaluates to $M(t) = e^{-\lambda} \cdot e^{\lambda e^t} = e^{\lambda(e^t - 1)}$ for all $t \in \mathbb{R}$.
    
18. Elementary Matrix Inverses:
    
    - Every elementary matrix is invertible, and its inverse is an elementary matrix of the exact same operational type.
    
19. Triangle Inequalities Proved:
    
    - Case 1 ($a+b \geq 0$): $\vert{}a+b\vert{} = a+b \leq \vert{}a\vert{} + \vert{}b\vert{}$ since $a \leq \vert{}a\vert{}$ and $b \leq \vert{}b\vert{}$.
    - Case 2 ($a+b < 0$): $\vert{}a+b\vert{} = -(a+b) = (-a) + (-b) \leq \vert{}a\vert{} + \vert{}b\vert{}$ since $-a \leq \vert{}a\vert{}$ and $-b \leq \vert{}b\vert{}$.
    - For the second target inequality: express $\vert{}a\vert{} = \vert{}(a-b) + b\vert{} \leq \vert{}a-b\vert{} + \vert{}b\vert{} \implies \vert{}a\vert{} - \vert{}b\vert{} \leq \vert{}a-b\vert{}$. Swapping the variables gives $\vert{}b\vert{} - \vert{}a\vert{} \leq \vert{}b-a\vert{} = \vert{}a-b\vert{} \implies -(\vert{}a\vert{} - \vert{}b\vert{}) \leq \vert{}a-b\vert{}$. Combining both directional inequalities yields $\vert{}\vert{}a\vert{} - \vert{}b\vert{}\vert{} \leq \vert{}a - b\vert{}$.
    
20. Range Space $R(T)$ Subspace Proof:
    
    - (1) Since $T(0) = 0$, $0 \in R(T)$.
    - (2) Let $w_1, w_2 \in R(T)$. Then $\exists v_1, v_2 \in V$ such that $T(v_1) = w_1$ and $T(v_2) = w_2$. By linearity, $w_1 + w_2 = T(v_1) + T(v_2) = T(v_1 + v_2)$. Since $v_1 + v_2 \in V$, $(w_1 + w_2) \in R(T)$.
    - (3) Let $c \in F$ and $w \in R(T)$ with $T(v) = w$. Then $cw = cT(v) = T(cv)$. Since $cv \in V$, $cw \in R(T)$. By Theorem 1.3, $R(T)$ is a valid subspace.
    
21. Cauchy-Schwarz Inequality Proof:
    
    - For any real $t$, consider the sum of squares $\sum_{i=1}^n (a_i t + b_i)^2 \geq 0$. Expanding this gives $A t^2 + 2B t + C \geq 0$, where $A = \sum a_i^2$, $B = \sum a_i b_i$, and $C = \sum b_i^2$. For this quadratic polynomial to remain non-negative for all real $t$, its discriminant must be less than or equal to zero: $(2B)^2 - 4AC \leq 4B^2 - 4AC \leq 0 \implies B^2 \leq AC$. Substituting back the sums yields $\left(\sum a_i b_i\right)^2 \leq \left(\sum a_i^2\right)\left(\sum b_i^2\right)$.
