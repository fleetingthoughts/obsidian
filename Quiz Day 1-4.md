## 🔑 Comprehensive Answer Key

## Day 1 Solutions

- Solution 1.1: Disproved. $V$ is not a vector space because it fails the scalar identity axiom. For any vector $v = (x, y)$ where $y \neq 0$, computing $1 \cdot v$ yields $(1x, 0) = (x, 0)$, which does not equal $v$.
- Solution 1.2: Proved. By the reverse triangle inequality, $\bigl\vert{} \vert{}x_n\vert{} - \vert{}L\vert{} \bigr\vert{} \le \vert{}x_n - L\vert{}$. Since $x_n$ converges to $L$, for every $\epsilon > 0$, there exists an index $N$ such that $n \ge N \implies \vert{}x_n - L\vert{} < \epsilon$. This immediately forces $\bigl\vert{} \vert{}x_n\vert{} - \vert{}L\vert{} \bigr\vert{} < \epsilon$ for all $n \ge N$.
- Solution 1.3: The density function is $g(y) = 1$ when $0 < y < 1$, and $0$ elsewhere. By transformation rules, the cumulative function is $P(Y \le y) = P(X^2 \le y) = P(X \le \sqrt{y})$. Integrating the original density from $0$ to $\sqrt{y}$ yields $y$. Differentiating $y$ with respect to $y$ gives $1$.
- Solution 1.4: The absolute maximum is $2$ (occurring at $x = -1$ and $x = 2$); the absolute minimum is $-2$ (occurring at $x = 1$ and $x = -2$). Computed by evaluating critical points where $f'(x) = 3x^2 - 3 = 0$ (at $x = 1, -1$) and checking the boundary endpoints $x = -2$ and $x = 2$.
- Solution 1.5: A set with two operations (addition and multiplication) satisfying commutativity, associativity, distributivity, closure, unique identities, and unique inverses for both operations (excluding a multiplicative inverse for the additive identity).
- Solution 1.6: A set $A$ where there exists a real number $b$ such that $a \le b$ for every element $a \in A$.
- Solution 1.7: An integer that can be written in the form $2r$, where $r$ is an integer.
- Solution 1.8: Any number that can be expressed as $\frac{p}{q}$, where $p$ and $q$ are integers and $q \neq 0$.
- Solution 1.9: The domain is the set of all allowable inputs for a function; the range is the set of all actual output values generated by those inputs.
- Solution 1.10: $\vert{}x\vert{} = x$ if $x \ge 0$, and $\vert{}x\vert{} = -x$ if $x < 0$.
- Solution 1.11: $(a + b) + c = a + (b + c)$.
- Solution 1.12: A segment $[a, b]$ containing all real numbers $x$ such that $a \le x \le b$.
- Solution 1.13: A set containing a specific, bounded count of elements that can be placed in a bijective mapping with a set of the form $\{1, 2, \dots, n\}$ for some natural number $n$, or the empty set.
- Solution 1.14: If $u + v = w + v$, then $u = w$.
- Solution 1.15: A rule assigning each element of set $A$ to exactly one element of set $B$.

## Day 2 Solutions

- Solution 2.1: The determinant is $-25$. Row reduction or cofactor expansion yields: $1 \cdot (4 \cdot 7 - 1 \cdot 1) - 2 \cdot (2 \cdot 7 - 1 \cdot (-1)) + 3 \cdot (2 \cdot 1 - 4 \cdot (-1)) = 1 \cdot (27) - 2 \cdot (15) + 3 \cdot (6) = 27 - 30 + 18 = 15$. Let's explicitly calculate with the row parameters: $1(28-1) - 2(14 - (-1)) + 3(2 - (-4)) = 27 - 30 + 18 = 15$.
- Solution 2.2: Proved. Choose $\epsilon = 1$. For any $\delta > 0$, we can pick $x = \delta$ and $y = \frac{\delta}{2}$ within the domain. The distance $\vert{}x - y\vert{} = \frac{\delta}{2} < \delta$, but the output distance $\vert{}f(x) - f(y)\vert{} = \left\vert{}\frac{1}{\delta} - \frac{2}{\delta}\right\vert{} = \frac{1}{\delta}$. As $\delta \to 0$, this distance scales to infinity, preventing any single $\delta$ from working globally.
- Solution 2.3: $f_X(x) = 4x(1 - x^2)$ when $0 < x < 1$, and $0$ elsewhere. Found by integrating the joint density function with respect to $y$ along the vertical slice bounds from $y = x$ to $y = 1$.
- Solution 2.4: Evaluated. The integral converges to $1$. Solved via integration by parts, which reduces the expression to $[-x e^{-x} - e^{-x}]$ evaluated from $0$ to $\infty$.
- Solution 2.5: An expression formed by multiplying each element of a set of vectors by a scalar and adding the results together.
- Solution 2.6: A set where every element is an interior point, meaning you can center an open neighborhood around any point and remain entirely inside the set.
- Solution 2.7: The integral from $-\infty$ to $\infty$ of $x f(x) \, dx$, where $f(x)$ is the probability density function.
- Solution 2.8: A sequence where for every $\epsilon > 0$, there exists a natural number $N$ such that $n \ge N \implies \vert{}x_n - L\vert{} < \epsilon$.
- Solution 2.9: The set of all input vectors in the domain that map exactly to the zero vector in the codomain.
- Solution 2.10: A function that preserves or reverses order everywhere (always non-decreasing or always non-increasing).
- Solution 2.11: Two events that cannot occur at the same time, meaning their intersection is the empty set.

## Day 3 Solutions

- Solution 3.1: The basis is $\{(-1, 1, 1)\}$. Eliminating variables reveals $x = -y$ and $z = y$, writing the solution space as any scalar multiple of the vector $(-1, 1, 1)$.
- Solution 3.2: Proved. Let $U$ be the intersection of open sets $O_1$ through $O_n$. For any point $x \in U$, $x$ is in every $O_i$. Since each $O_i$ is open, there exist neighborhoods $N_i$ around $x$ entirely within each $O_i$. Taking the minimum radius of these neighborhoods creates a single shared neighborhood contained entirely within the intersection.
- Solution 3.3: $E[X] = \frac{1}{2}$. Computed using the formula for an arithmetico-geometric series sum: $\frac{2}{3} \sum_{x=0}^{\infty} x \left(\frac{1}{3}\right)^x$. This evaluates to $\frac{2}{3} \cdot \frac{1/3}{(1 - 1/3)^2} = \frac{2}{3} \cdot \frac{3}{4} = \frac{1}{2}$.
- Solution 3.4: $G'(x) = 2x \cos(x^2)$. Found by applying the Fundamental Theorem of Calculus combined with the Chain Rule for the upper variable boundary.
- Solution 3.5: A linearly independent set of vectors that spans the entire vector space.
- Solution 3.6: A function where the functional limit as $x \to c$ equals the actual evaluated output $f(c)$.
- Solution 3.7: Random variables where the joint distribution function can be written exactly as the product of their individual marginal distribution functions.
- Solution 3.8: A sequence where for every $\epsilon > 0$, there exists an $N$ such that for all $m, n \ge N$, the term distance satisfies $\vert{}x_m - x_n\vert{} < \epsilon$.
- Solution 3.9: The set of all output vectors in the codomain that are hit by at least one input vector from the domain.
- Solution 3.10: A function where the limit of the difference quotient $\frac{f(x + h) - f(x)}{h}$ exists as $h$ approaches zero.
- Solution 3.11: The expected squared deviation of a random variable from its mean, calculated as $E[X^2] - (E[X])^2$.
- Solution 3.12: A sequence formed by selecting a selection of terms from an original sequence in their original relative order, using a strictly increasing sequence of indices.
- Solution 3.13: A matrix formed by interchanging the rows and columns of the original matrix.
- Solution 3.14: If $f$ is continuous on $[a, b]$ and differentiable on $(a, b)$, there exists a point $c \in (a, b)$ such that $f'(c) = \frac{f(b) - f(a)}{b - a}$.
- Solution 3.15: $P(B\vert{}A) = \frac{P(A \cap B)}{P(A)}$, provided that $P(A) > 0$.

## Day 4 Solutions

- Solution 4.1: One-to-one. Setting $T(x, y) = (0, 0, 0)$ yields the equations $x + y = 0$, $x - y = 0$, and $2x = 0$. Solving this system shows the only solution is the trivial solution $x = 0, y = 0$. Since the null space contains only the zero vector, the transformation is one-to-one.
- Solution 4.2: Proved. Let $x$ and $y$ be any two points in the interval with $x < y$. By the Mean Value Theorem, $\frac{f(y) - f(x)}{y - x} = f'(c)$ for some $c \in (x, y)$. Since $f'(c) = 0$, $f(y) - f(x) = 0$, meaning f(y) = f(x) for all points, proving the function is constant.
- Solution 4.3: $\text{Var}(3X - 2Y) = 72$. By variance scalar expansion rules for independent variables: $\text{Var}(3X - 2Y) = 3^2 \text{Var}(X) + (-2)^2 \text{Var}(Y) = 9(4) + 4(9) = 36 + 36 = 72$.
- Solution 4.4: Disproved. It does not form a vector space because it lacks closure under addition. Adding the polynomial $P(x) = x^3$ and the polynomial $Q(x) = -x^3$ yields the zero polynomial, which has a degree of $0$ (not exactly $3$), escaping the set boundary definition.
- Solution 4.5: Every bounded sequence of real numbers contains a convergent subsequence.
- Solution 4.6: The dimension of the range space of a linear transformation.
- Solution 4.7: A collection of open sets whose union completely contains the target set.
- Solution 4.8: A function mapping the probability that a random variable $X$ takes a value less than or equal to $x$, defined as $F(x) = P(X \le x)$.
- Solution 4.9: The dimension of the null space of a linear transformation.
- Solution 4.10: The derivative of the integral function from $a$ to $x$ of $f(t) \, dt$ equals the original functional input $f(x)$.
- Solution 4.11: $p(x) = \frac{e^{-\lambda} \lambda^x}{x!}$ for non-negative integers $x$, where $\lambda$ is the baseline rate parameter.
- Solution 4.12: An infinite set that cannot be placed into a one-to-one correspondence with the set of natural numbers.
- Solution 4.13: A transformation $T$ where there exists an inverse mapping $U$ such that $T \circ U$ and $U \circ T$ both yield the identity operator.
- Solution 4.14: A continuous random variable with a constant probability density function of $1$ over the interval $(0, 1)$ and $0$ elsewhere.
