We now depart from pure mathematics and look at applications of the exponential function. Courant proposes an alternative definition of the exponential function:

***Alternative definition of $f(x)=e^x$.*** If a function $y=f(x)$ satisfies an equation of the following form, then it is an exponential function: $$y'=\alpha y$$
$y= e^{\alpha x}$ sufficiently proves the differential equation but it can be shown that any function satisfying the differential equation $y'=\alpha y$  must be an exponential function as follows:
1) Assume $\exists$ a function $f$ that satisfies the differential equation
2) The "rabbit out of the hat step": Then it must be true that $fe^{\alpha x}$ is a constant because its derivative is necessarily 0 since both $f$ and $e^{\alpha x}$
3) Therefore $f$ must be an exponential as well for $fe^{\alpha x}$ to be a constant
