We now depart from pure mathematics and look at applications of the exponential function. Courant proposes an alternative definition of the exponential function:

***Alternative definition of $f(x)=e^x$.*** If a function $y=f(x)$ satisfies an equation of the following form, then it is an exponential function: $$y'=\alpha y$$
$y= e^{\alpha x}$ sufficiently proves the differential equation but it can be shown the converse is true by showing any function $y = fe^{-\alpha x}$ for some function $f$ collapses into a constant.
