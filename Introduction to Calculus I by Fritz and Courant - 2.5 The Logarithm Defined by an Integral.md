We now apply the concept of using an integral as a function to define special functions. Namely we define the logarithm and furnish properties of the logarithm. 
$$\int_a^b\frac{1}{u}du=\lim_{n \to \infty}\sqrt[n]{b/a-1}$$
We define the above limit as $\ln(x)$ with the lower limit $a=1$. Setting the lower limit this way lets us set $\ln(1)=0$ as a matter of convenience which lets us establish nice additive properties of logarithms. From this we can derive:

The addition theorem for any positive $x, y$ : $\ln(xy) = \ln(x)- \ln(y)$. This can be proven by looking directly at the limit definition or by Riemann sums
