## 📅 Day 1

## 📝 The Quiz

- Q1 (Real Analysis): State the Reverse Triangle Inequality for real numbers.
- Q2 (Linear Algebra): State the three conditions required to prove that a subset $W$ is a subspace of a vector space $V$.
- Q3 (Math Stats): State the strict definition of a Cumulative Distribution Function (CDF) for any random variable $X$.
- Q4 (Calculus): State the Chain Rule for the differentiation of a composite function.
- Q5 (Linear Algebra): Define what it means for a scalar $\lambda$ and a non-zero vector $v$ to be an eigenvalue and eigenvector of a linear operator $T$.

---

## 🔑 The Answer Key

## Q1 (Real Analysis)

- Direct Answer: $||a| - |b|| \leq |a - b|$ for all $a, b \in \mathbb{R}$.
- Textbook Context: Abbott outlines this as a critical variation of the standard Triangle Inequality used to establish lower bounds for spatial distances.

## Q2 (Linear Algebra)

- Direct Answer: $0 \in W$, and $W$ is closed under addition and scalar multiplication.
- Textbook Context: Friedberg explains that because general vector properties automatically inherit down to subsets, verifying these three closure rules is completely sufficient to declare a subspace configuration.

## Q3 (Math Stats)

- Direct Answer: $F(x) = P(X \leq x)$ for all real numbers $x$.
- Textbook Context: Hogg notes that a CDF maps out the entire cumulative probability accumulation profile of a random variable across the real line.

## Q4 (Calculus)

- Direct Answer: $\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)$.
- Textbook Context: Courant establishes this rule to break down the compound rate of change across chained variables into a product of local derivative steps.

## Q5 (Linear Algebra)

- Direct Answer: $T(v) = \lambda v$ where $v \neq 0$.
- Textbook Context: Friedberg states that an eigenvector identifies a characteristic spatial direction where the transformation acts as a simple scalar stretch.

## 📅 Day 2

## 📝 The Quiz

- Q1 (Real Analysis): State the precise definition of a countable set.
- Q2 (Linear Algebra): Define exactly what it means for a subset of vectors $S$ to be linearly independent.
- Q3 (Math Stats): State the mathematical distinction between counting permutations versus combinations.
- Q4 (Calculus): State the Fundamental Theorem of Calculus concerning the derivative of a functional accumulation area.
- Q5 (Math Stats): State the mathematical definition for the structural independence of two events $A$ and $B$.

---

## 🔑 The Answer Key

## Q1 (Real Analysis)

- Direct Answer: A set $A$ is countable if $\mathbb{N} \sim A$.
- Textbook Context: Abbott states that a set is countable if its elements can be put into a 1–1 correspondence with the natural numbers, essentially matching them to an infinitely long list.

## Q2 (Linear Algebra)

- Direct Answer: $\sum a_i u_i = 0 \implies a_i = 0$ for all $i$.
- Textbook Context: Friedberg defines linear independence as a state where no vector inside the subset can be constructed as a linear combination of the remaining elements.

## Q3 (Math Stats)

- Direct Answer: Permutations track order; combinations ignore it.
- Formulas: $P_k^n = \frac{n!}{(n-k)!}$ versus $\binom{n}{k} = \frac{n!}{k!(n-k)!}$.
- Textbook Context: Hogg frames combinations as counting distinct subsets, whereas permutations count distinct ordered arrangements drawn from an asset pool.

## Q4 (Calculus)

- Direct Answer: $\frac{d}{dx} \left[ \int_{a}^{x} f(t) \, dt \right] = f(x)$.
- Textbook Context: Courant positions this theorem to show that differentiation acts as the exact continuous inverse operation to accumulation.

## Q5 (Math Stats)

- Direct Answer: $P(A \cap B) = P(A)P(B)$.
- Textbook Context: Hogg demonstrates that this multiplication rule holds if and only if the occurrence of event $A$ provides absolutely zero predictive information about the likelihood of event $B$.

## 📅 Day 3

## 📝 The Quiz

- Q1 (Real Analysis): State the formal $\epsilon$-$N$ definition for the limit of a sequence.
- Q2 (Linear Algebra): State the Dimension Theorem (Rank-Nullity Theorem) for finite-dimensional vector spaces.
- Q3 (Math Stats): State the probability mass function (PMF), mean, and variance of a Poisson Distribution.
- Q4 (Calculus): State the formula for the derivative of an inverse function $f^{-1}(y)$.
- Q5 (Real Analysis): State the definition of an open set using the framework of neighborhoods.

---

## 🔑 The Answer Key

## Q1 (Real Analysis)

- Direct Answer: $\forall \epsilon > 0, \exists N \in \mathbb{N}$ such that $\forall n \geq N$, $|a_n - L| < \epsilon$.
- Textbook Context: Abbott emphasizes that the indexing tail $N$ forces the remaining infinite path of the sequence to sit entirely inside an $\epsilon$-neighborhood around the limit point $L$.

## Q2 (Linear Algebra)

- Direct Answer: $\dim(V) = \dim(N(T)) + \dim(R(T))$.
- Textbook Context: Friedberg tracks this identity to prove that an operator splits the domain's dimensions cleanly between collapsed elements (Null Space) and preserved outputs (Range).

## Q3 (Math Stats)

- Direct Answer: $p(x) = \frac{e^{-\lambda}\lambda^x}{x!}$ for $x \in \{0,1,2,\dots\}$; $\mu = \lambda$, $\sigma^2 = \lambda$.
- Textbook Context: Hogg uses this distribution to model the occurrence rate of independent events falling within a fixed, continuous domain.

## Q4 (Calculus)

- Direct Answer: $(f^{-1})'(y_0) = \frac{1}{f'(x_0)}$ where $y_0 = f(x_0)$ and $f'(x_0) \neq 0$.
- Textbook Context: Courant demonstrates this geometrically: swapping your axis definitions naturally reciprocates the instantaneous tangent slopes along a graph.

## Q5 (Real Analysis)

- Direct Answer: A set where every point is an interior point.
- Textbook Context: Abbott defines a set $O \subseteq \mathbb{R}$ as open if for every element $x \in O$, there exists an $\epsilon$-neighborhood $V_\epsilon(x)$ entirely contained inside $O$.

## 📅 Day 4

## 📝 The Quiz

- Q1 (Real Analysis): State the Monotone Convergence Theorem for sequences.
- Q2 (Linear Algebra): Define how the entry elements of a matrix representation $[T]_\beta^\gamma$ are determined.
- Q3 (Math Stats): State the explicit definition of Convergence in Probability ($X_n \xrightarrow{p} X$).
- Q4 (Calculus): State the general formula for Integration by Parts on a bounded definite segment.
- Q5 (Linear Algebra): List the three geometric axioms required to define an Inner Product.

---

## 🔑 The Answer Key

## Q1 (Real Analysis)

- Direct Answer: A monotone sequence converges if and only if it is bounded.
- Textbook Context: Abbott showcases this as a direct consequence of the completeness of the real line, establishing that unidirectional bounded numbers cannot avoid hitting a limiting barrier.

## Q2 (Linear Algebra)

- Direct Answer: Columns are coordinate vectors of the transformed basis $\beta$ relative to basis $\gamma$.
- Textbook Context: Friedberg structures the $j$-th column of $[T]_\beta^\gamma$ as exactly $[T(v_j)]_\gamma$, binding abstract transformations directly to matrix algebra.

## Q3 (Math Stats)

- Direct Answer: $\lim_{n \to \infty} P(|X_n - X| \geq \epsilon) = 0$ for all $\epsilon > 0$.
- Textbook Context: Hogg leverages this asymptotic mode to verify _estimator consistency_, ensuring tracking errors collapse to zero as your data pool expands.

## Q4 (Calculus)

- Direct Answer: $\int_a^b u \, dv = [uv]_a^b - \int_a^b v \, du$.
- Textbook Context: Courant employs this formula to resolve complex product integrations by systematically reversing the differential product rule.

## Q5 (Linear Algebra)

- Direct Answer: Mappings must satisfy linearity, symmetry, and positive-definiteness.
- Textbook Context: Friedberg utilizes these criteria to introduce concrete geometric measures—such as lengths, metrics, and orthogonal angles—to abstract vector components.

## 📅 Day 5

## 📝 The Quiz

- Q1 (Real Analysis): State the Bolzano–Weierstrass Theorem for real sequences.
- Q2 (Linear Algebra): Define the Adjoint $T^*$ of a linear operator on an inner product space.
- Q3 (Math Stats): State the requirement for an estimator to be considered unbiased.
- Q4 (Calculus): State the Mean Value Theorem for derivatives.
- Q5 (Math Stats): State the analytical formula for Fisher Information $I(\theta)$ within a single observation.

---

## 🔑 The Answer Key

## Q1 (Real Analysis)

- Direct Answer: Every bounded sequence contains a convergent subsequence.
- Textbook Context: Abbott leverages this theorem to establish structural continuity: trapping an infinite path inside a closed environment forces elements to accumulate somewhere.

## Q2 (Linear Algebra)

- Direct Answer: The unique operator satisfying $\langle T(x), y \rangle = \langle x, T^*(y) \rangle$.
- Textbook Context: Friedberg tracks adjoint formulations because they allow linear operators to slide smoothly across inner product pairings.

## Q3 (Math Stats)

- Direct Answer: An estimator whose expected value matches the true parameter: $E[\hat{\theta}] = \theta$.
- Textbook Context: Hogg relies on this property to ensure that an estimator's distribution balances accurately over the target without systematic distortion.

## Q4 (Calculus)

- Direct Answer: $f'(c) = \frac{f(b) - f(a)}{b - a}$ for some $c \in (a,b)$.
- Textbook Context: Courant positions this theorem to verify that a smooth graph's continuous trajectory must contain a point where its local tangent mirrors the regional chord.

## Q5 (Math Stats)

- Direct Answer: $I(\theta) = E\left[\left(\frac{\partial}{\partial \theta} \log f(X;\theta)\right)^2\right] = -E\left[\frac{\partial^2}{\partial \theta^2} \log f(X;\theta)\right]$.
- Textbook Context: Hogg introduces this metric to evaluate the informational payload of a sample, forming the variance baseline for parameter estimation.

## 📅 Day 6

## 📝 The Quiz

- Q1 (Real Analysis): State the topological characterization of a compact set via the Heine–Borel Theorem.
- Q2 (Linear Algebra): State the definition of a diagonalizable linear operator.
- Q3 (Math Stats): State the Fisher–Neyman Factorization Theorem for parameter sufficiency.
- Q4 (Calculus): State the Mean Value Theorem for Integrals.
- Q5 (Real Analysis): State the definition of Uniform Continuity across a domain $A$.

---

## 🔑 The Answer Key

## Q1 (Real Analysis)

- Direct Answer: A subset of $\mathbb{R}$ is compact if and only if it is closed and bounded.
- Textbook Context: Abbott presents this characterization to show that compact sets let us reliably pass local analytical conditions across finite coverage networks.

## Q2 (Linear Algebra)

- Direct Answer: An operator whose space possesses a basis consisting entirely of eigenvectors.
- Textbook Context: Friedberg explains that diagonalizability decouples mixed linear dynamics into a clean, standalone system of scalar equations.

## Q3 (Math Stats)

- Direct Answer: $f(\mathbf{x};\theta) = g(u(\mathbf{x});\theta) \cdot h(\mathbf{x})$ implies sufficiency of $u(\mathbf{X})$.
- Textbook Context: Hogg notes that isolating the parameter link into a dedicated kernel function proves that the remaining data profile contains no further parameter insights.

## Q4 (Calculus)

- Direct Answer: $\int_a^b f(x) \, dx = f(c)(b - a)$ for some $c \in [a,b]$.
- Textbook Context: Courant notes that for any continuous graph, the accumulated area can always be matched by a flat rectangle evaluated at an intermediate point.

## Q5 (Real Analysis)

- Direct Answer: $\forall \epsilon > 0, \exists \delta > 0$ such that $\forall x, y \in A, |x - y| < \delta \implies |f(x) - f(y)| < \epsilon$.
- Textbook Context: Abbott highlights that uniform continuity locks down a single, global scaling control $\delta$ that operates independently of your position in the domain.

## 📅 Day 7

## 📝 The Quiz

- Q1 (Real Analysis): State the theorem regarding Uniform Convergence and Differentiation for a sequence of functions.
- Q2 (Linear Algebra): Define the Orthogonal Complement $W^\perp$ of a subspace.
- Q3 (Math Stats): State the classical univariate Central Limit Theorem (CLT).
- Q4 (Calculus): State the Fundamental Theorem of Calculus for evaluating definite integrals via primitives.
- Q5 (Math Stats): Define the errors classified as Type I Error and Type II Error.

---

## 🔑 The Answer Key

## Q1 (Real Analysis)

- Direct Answer: If $f_n \to f$ pointwise, $f_n'$ converges uniformly to $g$, then $f_n \to f$ uniformly and $f' = g$.
- Textbook Context: Abbott presents this theorem to outline the conditions required to safely swap limit operators and derivative steps without distorting structural calculations.

## Q2 (Linear Algebra)

- Direct Answer: $W^\perp = \{x \in V : \langle x, y \rangle = 0 \text{ for all } y \in W\}$.
- Textbook Context: Friedberg utilizes this complement to isolate independent geometric environments, cleanly partitioning vectors into orthogonal projections.

## Q3 (Math Stats)

- Direct Answer: $\frac{\bar{X}_n - \mu}{\sigma/\sqrt{n}} \xrightarrow{d} \mathcal{N}(0, 1)$ as $n \to \infty$ for i.i.d. variables with finite variance.
- Textbook Context: Hogg outlines this as a cornerstone of statistics, showing that large averages naturally track normal profiles regardless of the population's starting shape.

## Q4 (Calculus)

- Direct Answer: $\int_a^b f(x) \, dx = F(b) - F(a)$ where $F'(x) = f(x)$.
- Textbook Context: Courant establishes this as a major shortcut, bypassing tedious Riemann boundary summation limits through direct antiderivative evaluation.

## Q5 (Math Stats)

- Direct Answer: Type I rejects a true null hypothesis; Type II retains a false one.
- Textbook Context: Hogg notes that hypothesis testing balances these competing error types to structure robust boundaries for statistical decisions.
