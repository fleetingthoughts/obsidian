## 📅 Day 1

## Card 1 (Real Analysis)

- Front: State the classical Triangle Inequality for real numbers.
- Back:
    
    - Formula: $|a + b| \leq |a| + |b|$ for all $a, b \in \mathbb{R}$.
    - Abbott Context: The foundational inequality in real analysis used to establish upper bounds for the distance between accumulating functional pathways.
    

## Card 2 (Linear Algebra)

- Front: State the Cancellation Law for Vector Addition in an abstract vector space.
- Back:
    
    - Theorem: If $x, y, z$ are vectors in a vector space $V$ such that $x + z = y + z$, then $x = y$.
    - Friedberg Context: Proven directly by adding the unique additive inverse of $z$ (denoted $-z$) to both sides and applying associativity.
    

## Card 3 (Math Stats)

- Front: State the three foundational Axioms of Probability that define a probability set function.
- Back:
    
    - Axioms: For a sample space $\mathcal{C}$ and event collection $\mathcal{B}$:
        
        1. $P(A) \geq 0$ for all $A \in \mathcal{B}$.
        2. $P(\mathcal{C}) = 1$.
        3. If $\{An\}$ is a sequence of mutually exclusive events, then $P\left(\bigcup_{n=1}^{\infty} An\right) = \sum_{n=1}^{\infty} P(An)$.
        
    - Hogg Context: These mathematical conditions formalize the long-run stabilizing trends observed in empirical relative frequencies.
    

## Card 4 (Calculus)

- Front: State the Quotient Rule for the differentiation of two nested functions.
- Back:
    
    - Formula: $\frac{d}{dx}\left[\frac{f(x)}{g(x)}\right] = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}$, provided $g(x) \neq 0$.
    - Courant Context: Evaluates the simultaneous combinations of rate changes when a system is modeled as a ratio of changing quantities.

## 📅 Day 2

## Card 5 (Linear Algebra)

- Front: Prove or explain why the scalar multiplication property $0x = 0$ must hold for any vector $x$.
- Back:
    
    - Proof Fact: $0x + 0x = (0 + 0)x = 0x = 0x + 0$. By the Cancellation Law, $0x = 0$.
    - Friedberg Context: Establishes that multiplying any vector by the scalar field's zero element collapses the system directly into the vector space's zero element.
    

## Card 6 (Math Stats)

- Front: State the Complement Rule Theorem for the probability of an event's non-occurrence.
- Back:
    
    - Formula: $P(A) = 1 - P(A^c)$ for any event $A$.
    - Hogg Context: Derived rigorously because the sample space forms a disjoint partition between an event and its absolute complement: $\mathcal{C} = A \cup A^c$.
    

## Card 7 (Calculus)

- Front: State the power rule formula for the derivative of $f(x) = x^n$ where $n \in \mathbb{N}$.
- Back:
    
    - Formula: $\frac{d}{dx}[x^n] = n x^{n-1}$.
    - Courant Context: Foundational rate rule derived by expanding the difference quotient via the binomial theorem or algebraic factorization blocks.
    

## Card 8 (Real Analysis)

- Front: State the strict definition of a Supremum (Least Upper Bound) of a set $A \subseteq \mathbb{R}$.
- Back:
    
    - Definition: A real number $s$ is the supremum of $A$ ($s = \sup A$) if:
        
        1. $s$ is an upper bound ($a \leq s$ for all $a \in A$).
        2. If $b$ is any upper bound for $A$, then $s \leq b$.
        
    - Abbott Context: Represents the lowest possible ceiling that limits a numerical set profile without requiring the ceiling value to belong to the set itself.

## 📅 Day 3

## Card 9 (Math Stats)

- Front: State the general Addition Rule for the probability of the union of any two events.
- Back:
    
    - Formula: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$.
    - Hogg Context: Adjusts the direct sum of the individual probabilities by subtracting the overlapping intersection region to prevent double-counting.
    

## Card 10 (Calculus)

- Front: Define what distinguishes an even function from an odd function mathematically.
- Back:
    
    - Definitions:
        
        - Even: $f(-x) = f(x)$ (symmetric across the $y$-axis).
        - Odd: $f(-x) = -f(x)$ (symmetric across the origin coordinates).
        
    - Courant Context: Recognizing these structural internal symmetries simplifies the integration of complex curves over symmetric intervals $[-a, a]$.
    

## Card 11 (Real Analysis)

- Front: State Lemma 1.3.8 (the $\epsilon$-characterization of a Least Upper Bound).
- Back:
    
    - Lemma: Assume $s$ is an upper bound for $A$. Then $s = \sup A$ if and only if for every $\epsilon > 0$, there exists an element $a \in A$ satisfying $s - \epsilon < a$.
    - Abbott Context: Proves that dropping your upper bound by even an infinitesimal step $\epsilon$ guarantees you will cross into the set, tracking its upper boundary.
    

## Card 12 (Linear Algebra)

- Front: State the definition of a Linear Combination of vectors.
- Back:
    
    - Definition: A vector $v \in V$ is a linear combination of vectors $u_1, u_2, \dots, u_n \in V$ if there exist scalars $a_1, a_2, \dots, a_n \in F$ such that $v = a_1 u_1 + a_2 u_2 + \dots + a_n u_n$.
    - Friedberg Context: The fundamental synthesis operation in linear algebra, mixing scaled directional pathways together to navigate across a spatial coordinate framework.

## 📅 Day 4

## Card 13 (Calculus)

- Front: Define the absolute components of a functional mapping: Domain versus Range.
- Back:
    
    - Definitions:
        
        - Domain: The complete set of valid independent input variables $x$.
        - Range: The complete set of dependent functional output values $f(x)$ generated from the input inputs.
        
    - Courant Context: Delineates the geometric boundaries where a model safely operates along coordinate planes.
    

## Card 14 (Real Analysis)

- Front: State the Archimedean Property variation concerning fractional limits.
- Back:
    
    - Theorem: Given any real number $y > 0$, there exists a natural number $n \in \mathbb{N}$ satisfying $\frac{1}{n} < y$.
    - Abbott Context: Proves that the real system does not contain infinitely small elements; you can choose a denominator large enough to shrink a fractional step below any positive baseline.
    

## Card 15 (Linear Algebra)

- Front: State the definition of the Span of a subset of vectors $S$.
- Back:
    
    - Definition: The span of $S$, denoted $\text{span}(S)$, is the collection of all possible finite linear combinations of the vectors in $S$.
    - Friedberg Context: Geometrically maps out the entire structural subspace swept out and generated by a collection of baseline directional items.
    

## Card 16 (Math Stats)

- Front: Define the criteria for the Equilikely Case on a finite sample space.
- Back:
    
    - Definition: If a sample space has $m$ elements, every individual point has probability $p_i = \frac{1}{m}$. The probability of any event $A$ simplifies to $P(A) = \frac{\#(A)}{m}$.
    - Hogg Context: The mathematical backbone for classical gambling and drawing calculations, requiring only accurate counting parameters to output exact probabilities.

## 📅 Day 5

## Card 17 (Real Analysis)

- Front: State the Density Property of $\mathbb{Q}$ in $\mathbb{R}$.
- Back:
    
    - Theorem: For any two real numbers $a, b$ with $a < b$, there exists a rational number $r \in \mathbb{Q}$ satisfying $a < r < b$.
    - Abbott Context: Demonstrates that the rational numbers are thoroughly mixed into the real axis, ensuring a fraction can be squeezed between any two distinct locations.
    

## Card 18 (Linear Algebra)

- Front: Define the Trace of a square matrix.
- Back:
    
    - Definition: The trace of an $n \times n$ matrix $M$, denoted $\text{tr}(M)$, is the sum of its diagonal entries: $\text{tr}(M) = \sum_{i=1}^n M_{ii}$.
    - Friedberg Context: A linear scalar metric that remains perfectly invariant even when you completely change the surrounding coordinate frame or bases.
    

## Card 19 (Math Stats)

- Front: State the probability calculation logic for the classic Birthday Problem with $n$ people.
- Back:
    
    - Formula: $P(\text{at least one match}) = 1 - \frac{P_{365}^n}{365^n}$.
    - Hogg Context: Solved efficiently by calculating the complement probability: counting the ordered arrangements where all $n$ birthdays are strictly distinct.
    

## Card 20 (Calculus)

- Front: State the formal definition of Left-hand and Right-hand limits at a point.
- Back:
    
    - Definitions:
        
        - Left: $\lim_{x \to c^-} f(x) = L$ evaluates approach paths strictly where $x < c$.
        - Right: $\lim_{x \to c^+} f(x) = L$ evaluates approach paths strictly where $x > c$.
        
    - Courant Context: A functional limit exists globally if and only if these two directional limits converge onto the exact same value.

## Card 21 (Linear Algebra)

- Front: State the uniqueness corollary regarding the Zero Vector of a space.
- Back:
    
    - Corollary: The additive identity vector described in vector space axiom (VS3) is strictly unique.
    - Friedberg Context: Proven by assuming two identities $0$ and $0'$, which forces $0 = 0 + 0' = 0'$, locking down a single unique zero reference point.
    

## Card 22 (Math Stats)

- Front: State the probability mass function (PMF), mean, and variance of a Binomial Distribution.
- Back:
    
    - Formula: $p(x) = \binom{n}{x}p^x(1-p)^{n-x}$ for $x \in \{0, 1, \dots, n\}$.
    - Parameters: Mean $\mu = np$, Variance $\sigma^2 = np(1-p)$.
    - Hogg Context: Models the cumulative count of independent success events tracked over a fixed string of $n$ trials.
    

## Card 23 (Calculus)

- Front: State the analytical definition of functional continuity at a point using limit notation.
- Back:
    
    - Definition: A function $f(x)$ is continuous at a point $c$ if $\lim_{x \to c} f(x) = f(c)$.
    - Courant Context: Requires the local limit to exist, the function to be defined at that coordinate, and both values to match exactly, preventing gaps or jumps.
    

## Card 24 (Real Analysis)

- Front: State the strict definition of an Injective (1-1) function versus a Surjective (onto) function.
- Back:
    
    - Definitions:
        
        - Injective: $f(a_1) = f(a_2) \implies a_1 = a_2$ (no two inputs share an output).
        - Surjective: For every $b \in B$, there exists an $a \in A$ such that $f(a) = b$ (the range fills the codomain).
        
    - Abbott Context: Structural parameters used to establish absolute structural mappings and size equivalences across infinite domains.

## 📅 Day 7

## Card 25 (Math Stats)

- Front: State the probability mass function (PMF), mean, and variance of a Geometric Distribution.
- Back:
    
    - Formula: $p(x) = p(1-p)^{x-1}$ for $x \in \{1, 2, 3, \dots\}$.
    - Parameters: Mean $\mu = \frac{1}{p}$, Variance $\sigma^2 = \frac{1-p}{p^2}$.
    - Hogg Context: Models the waiting time or number of independent trials required to witness the very first success event.
    

## Card 26 (Calculus)

- Front: State the definition and physical interpretation of a Second Derivative.
- Back:
    
    - Definition: $f''(x) = \frac{d}{dx}[f'(x)]$.
    - Courant Context: Measures the rate of change of the slope itself, tracking the geometric curvature (concurrence/concavity) or the physical acceleration of a moving object.
    

## Card 27 (Real Analysis)

- Front: State the structural $\epsilon$-$\delta$ definition of functional continuity at a point $c$.
- Back:
    
    - Definition: $f$ is continuous at $c$ if $\forall \epsilon > 0, \exists \delta > 0$ such that $|x - c| < \delta \implies |f(x) - f(c)| < \epsilon$.
    - Abbott Context: Formalizes the notion that keeping your input coordinate tightly bound within a distance $\delta$ guarantees your output stays locked inside an error neighborhood $\epsilon$.
    

## Card 28 (Linear Algebra)

- Frontonomy: State the definition and formula for the Additive Inverse vector.
- Back:
    
    - Definition: For every element $x \in V$, there exists a unique vector $y \in V$ such that $x + y = 0$. This vector is denoted uniquely as $-1x = -x$.
    - Friedberg Context: Guaranteed by axiom (VS4) to secure a direction-reversing vector element, providing the structural basis needed to perform vector subtraction.
